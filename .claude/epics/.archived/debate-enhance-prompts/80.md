# Task: Update Risk agent — integrate appendix, strategy tree, and deduplicate validator

## Description
Update `src/options_arena/agents/risk.py` to:
1. Append the shared `PROMPT_RULES_APPENDIX` to its system prompt
2. Add a new `RISK_STRATEGY_TREE` constant with the strategy selection decision tree
3. Append `RISK_STRATEGY_TREE` to the risk system prompt

on top of the shared calibration/citation rules.
