# Task: Write tests for shared helpers and prompt integration

## Description
Write ~6 unit tests covering the new shared prompt appendix, output validator helpers, and
their integration into each agent's system prompt. Then run the full verification gate
(ruff, pytest, mypy) to confirm all changes are clean.

## Acceptance Criteria
- [ ] Test: `PROMPT_RULES_APPENDIX` is present in bull agent's effective system prompt
- [ ] Test: `PROMPT_RULES_APPENDIX` is present in bear agent's effective system prompt
- [ ] Test: `PROMPT_RULES_APPENDIX` is present in risk agent's effective system prompt
- [ ] Test: `RISK_STRATEGY_TREE` is present in risk agent's prompt ONLY (not bull/bear)
- [ ] Test: `build_cleaned_agent_response()` strips `<think>` tags from all text fields
- [ ] Test: `build_cleaned_trade_thesis()` strips `<think>` tags from all text fields
- [ ] Test: `build_cleaned_agent_response()` returns original when no `<think>` tags
- [ ] Test: `build_cleaned_trade_thesis()` returns original when no `<think>` tags
- [ ] Test: Version headers `v2.0` on all three agent prompts
- [ ] All 1,262+ existing tests still pass
- [ ] Full verification gate passes: ruff check, ruff format, pytest, mypy --strict

## Technical Details
- **Test file**: `tests/agents/test_prompt_enhancements.py` (new file)
- **Testing pattern**: Use `models.ALLOW_MODEL_REQUESTS = False` at module level
- **Prompt inspection**: Use `TestModel` + `agent.override()` to inspect effective prompts,
  OR directly check module-level constants (`BULL_SYSTEM_PROMPT`, `BEAR_SYSTEM_PROMPT`,
  `RISK_SYSTEM_PROMPT`) since appendix is concatenated at module level
- **Helper tests**: Direct function calls with mock `AgentResponse`/`TradeThesis` instances
  - Create test instances with `<think>reasoning</think>` in text fields
  - Verify cleaned output has no `<think>` tags
  - Verify original is returned when no tags present (identity check: `output is result`)
- **Fixtures needed**:
  - `mock_agent_response_with_think_tags` — `AgentResponse` with `<think>` in argument/key_points
  - `mock_agent_response_clean` — `AgentResponse` with no `<think>` tags
  - `mock_trade_thesis_with_think_tags` — `TradeThesis` with `<think>` in summary/key_factors
  - `mock_trade_thesis_clean` — `TradeThesis` with no `<think>` tags
