# Task: Analysis, scan & health models

## Description
Implement three model files: `analysis.py` (MarketContext, AgentResponse, TradeThesis), `scan.py` (IndicatorSignals, ScanRun, TickerScore), and `health.py` (HealthStatus).

## Acceptance Criteria
- [ ] `src/options_arena/models/analysis.py` contains:
  - `MarketContext` — flat model (not nested), fields per CLAUDE.md spec including `dividend_yield: float`, `exercise_style: ExerciseStyle`, `data_timestamp: datetime`
  - `AgentResponse` — frozen=True, `confidence` validated [0.0, 1.0]
  - `TradeThesis` — frozen=True, with `recommended_strategy: SpreadType | None = None`
  - Decimal field_serializers on MarketContext price fields
- [ ] `src/options_arena/models/scan.py` contains:
  - `IndicatorSignals` — 18 named `float | None` fields (NOT frozen, populated incrementally)
  - `ScanRun` — frozen=True, fields: `id`, `started_at`, `completed_at`, `preset`, `tickers_scanned`, `tickers_scored`, `recommendations`
  - `TickerScore` — fields: `ticker`, `composite_score`, `direction: SignalDirection`, `signals: IndicatorSignals`, `scan_run_id`
- [ ] `src/options_arena/models/health.py` contains:
  - `HealthStatus` — frozen=True, fields: `service_name`, `available`, `latency_ms`, `error`, `checked_at`
- [ ] `IndicatorSignals` has exactly 18 fields, all `float | None = None`
- [ ] `IndicatorSignals()` with no args constructs successfully (all None)
- [ ] Passes `ruff check` and `mypy --strict`

## Technical Details
- `MarketContext` is flat — agents parse flat text better than nested objects
- `IndicatorSignals` is NOT frozen — gets populated incrementally during the scan pipeline
- `TickerScore.signals` is `IndicatorSignals` (typed model), never `dict[str, float]`
- `AgentResponse` and `TradeThesis` are shapes for v2 debate — define now, use later
- All 18 indicator field names must match exactly: rsi, stochastic_rsi, williams_r, adx, roc, supertrend, bb_width, atr_pct, keltner_width, obv, ad, relative_volume, sma_alignment, vwap_deviation, iv_rank, iv_percentile, put_call_ratio, max_pain_distance
