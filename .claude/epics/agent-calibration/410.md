# Task: CLI Commands — agent-accuracy, calibration, agent-weights

## Description

Add 3 new subcommands to the `outcomes` Typer sub-app: `agent-accuracy`, `calibration`,
and `agent-weights`. Each follows the existing pattern of sync Typer command wrapping
`asyncio.run()` with Rich table output.

## Acceptance Criteria

- [ ] `outcomes agent-accuracy [--window DAYS]` — Rich table: Agent, Hit Rate, Confidence, Brier, Samples
- [ ] `outcomes calibration [--agent NAME]` — Rich table: per-agent or aggregate calibration buckets
- [ ] `outcomes agent-weights` — Rich table: Agent, Manual, Auto-Tuned, Brier, Samples
- [ ] All commands are sync Typer with `asyncio.run()` wrapper
- [ ] Graceful handling when no data available (informative message, not crash)
- [ ] `--window` accepts positive integer (default: all-time)
- [ ] `--agent` accepts agent name string (default: aggregate view)
- [ ] Float formatting: percentages to 1 decimal, Brier to 3 decimals
- [ ] `isfinite()` check before formatting — fall back to `"--"` for non-finite

### Pattern (from existing commands)
```python
@outcomes_app.command("agent-accuracy")
def agent_accuracy_cmd(
    window: Annotated[int | None, typer.Option("--window", help="Rolling window in days")] = None,
) -> None:
    """Show per-agent direction accuracy and Brier scores."""
    asyncio.run(_agent_accuracy_async(window))

async def _agent_accuracy_async(window: int | None) -> None:
    db = Database(DB_PATH)
    await db.initialize()
    repo = Repository(db)
    try:
        results = await repo.get_agent_accuracy(window)
        if not results:
            console.print("[yellow]No agent accuracy data available (need 10+ outcomes).[/yellow]")
            return
        table = Table(title="Agent Accuracy")
        table.add_column("Agent", style="cyan")
        table.add_column("Hit Rate", justify="right")
        table.add_column("Confidence", justify="right")
        table.add_column("Brier", justify="right")
        table.add_column("Samples", justify="right", style="dim")
        for r in results:
            table.add_row(
                r.agent_name,
                f"{r.direction_hit_rate:.1%}" if math.isfinite(r.direction_hit_rate) else "--",
                f"{r.mean_confidence:.1%}" if math.isfinite(r.mean_confidence) else "--",
                f"{r.brier_score:.3f}" if math.isfinite(r.brier_score) else "--",
                str(r.sample_size),
            )
        console.print(table)
    finally:
        await db.close()
```

Similar patterns for `calibration` and `agent-weights`.
