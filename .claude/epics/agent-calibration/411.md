## Technical Details

### Models (in `models/analytics.py`, after `PerformanceSummary` ~line 601)

```python
class AgentAccuracyReport(BaseModel):
    model_config = ConfigDict(frozen=True)
    agent_name: str
    direction_hit_rate: float  # 0.0-1.0
    mean_confidence: float     # 0.0-1.0
    brier_score: float         # 0.0-1.0 (lower = better)
    sample_size: int

class CalibrationBucket(BaseModel):
    model_config = ConfigDict(frozen=True)
    bucket_label: str          # e.g. "0.0-0.2"
    bucket_low: float
    bucket_high: float
    mean_confidence: float
    actual_hit_rate: float
    count: int

class AgentCalibrationData(BaseModel):
    model_config = ConfigDict(frozen=True)
    agent_name: str | None     # None = aggregate
    buckets: list[CalibrationBucket]
    sample_size: int

class AgentWeightsComparison(BaseModel):
    model_config = ConfigDict(frozen=True)
    agent_name: str
    manual_weight: float
    auto_weight: float
    brier_score: float | None  # None if < 10 samples
    sample_size: int
```

### Migrations

`data/migrations/026_agent_accuracy_index.sql`:
```sql
CREATE INDEX IF NOT EXISTS idx_ap_name_created ON agent_predictions(agent_name, created_at);
```

`data/migrations/027_auto_tune_weights.sql`:
```sql
CREATE TABLE IF NOT EXISTS auto_tune_weights (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    agent_name TEXT NOT NULL,
    manual_weight REAL NOT NULL,
    auto_weight REAL NOT NULL,
    brier_score REAL,
    sample_size INTEGER NOT NULL,
    window_days INTEGER NOT NULL,
    created_at TEXT NOT NULL
);
CREATE INDEX IF NOT EXISTS idx_atw_created ON auto_tune_weights(created_at);
```
