# Task: Weight Computation + Orchestrator Integration

## Description

Implement the auto-tune weight computation algorithm as a pure function, then integrate it
into the orchestrator. `synthesize_verdict()` gets an optional `vote_weights` parameter.
`run_debate()` loads auto-tuned weights when `config.auto_tune_weights=True`.

## Acceptance Criteria

- [ ] `compute_auto_tune_weights(accuracy: list[AgentAccuracyReport]) -> dict[str, float]` — pure function
- [ ] Agents with < 10 samples keep manual weight from `AGENT_VOTE_WEIGHTS`
- [ ] `raw_weight = 1.0 - brier_score` for agents with ≥ 10 samples
- [ ] Floor=0.05, cap=0.35 per agent; risk always 0.0
- [ ] Directional weights normalized to sum=0.85 (Bordley 1982)
- [ ] `math.isfinite()` on all computed weights
- [ ] `synthesize_verdict()` accepts `vote_weights: dict[str, float] | None = None`
- [ ] When `vote_weights` is None, behavior is identical to current (backward compatible)
- [ ] `run_debate()` loads auto-tuned weights when `config.auto_tune_weights=True` and `repository` provided
- [ ] All existing orchestrator tests pass unchanged

## Technical Details

### Weight Computation Function

Location: `src/options_arena/agents/orchestrator.py` (near `AGENT_VOTE_WEIGHTS`)

```python
def compute_auto_tune_weights(
    accuracy: list[AgentAccuracyReport],
) -> dict[str, float]:
    """Compute auto-tuned vote weights from agent accuracy data.

    Uses inverse Brier score, normalized to sum=0.85.
    Agents with <10 samples keep manual weights.
    """
    weights = dict(AGENT_VOTE_WEIGHTS)  # start with manual
    agents_with_data = {r.agent_name: r for r in accuracy if r.sample_size >= 10}

    for name in weights:
        if name == "risk":
            weights[name] = 0.0
            continue
        if name in agents_with_data:
            weights[name] = 1.0 - agents_with_data[name].brier_score

    # Floor and cap
    for name in weights:
        if name == "risk":
            continue
        weights[name] = max(0.05, min(0.35, weights[name]))

    # Normalize directional weights to 0.85
    directional = {k: v for k, v in weights.items() if k != "risk"}
    total = sum(directional.values())
    if total > 0:
        for name in directional:
            weights[name] = (directional[name] / total) * 0.85

    return weights
```

### Edge Cases
- All agents have Brier score 0.0 (perfect) → equal weights, sum=0.85
- All agents have Brier score 1.0 (worst) → floor weights, renormalized
- Mixed: some agents have data, some don't → hybrid manual + computed
- Single agent has data, others don't → that agent's weight bounded by cap
