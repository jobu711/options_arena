# Task: Bull, Bear, and Risk agents

## Description

Create all three PydanticAI agent modules: `bull.py`, `bear.py`, `risk.py`. Each agent is
a module-level `Agent[DebateDeps, OutputType]` instance with a system prompt constant,
`@system_prompt` decorator, and `@output_validator` for `<think>` tag rejection.

Agents have NO knowledge of each other. Each imports only from `_parsing.py` and `models/`.

- [ ] `agents/bull.py` created with `bull_agent: Agent[DebateDeps, AgentResponse]`
- [ ] `agents/bear.py` created with `bear_agent: Agent[DebateDeps, AgentResponse]`
- [ ] `agents/risk.py` created with `risk_agent: Agent[DebateDeps, TradeThesis]`
- [ ] All agents use `model=None` at init, `deps_type=DebateDeps`, `retries=2`

- [ ] Risk uses `@risk_agent.system_prompt(dynamic=True)` — injects both bull and bear arguments
- [ ] All three have `@agent.output_validator` that rejects `<think>` / `</think>` tags via `ModelRetry`

## Technical Details

### Agent Pattern (all three follow this)

```python
from pydantic_ai import Agent, ModelRetry, RunContext
from pydantic_ai.settings import ModelSettings
from options_arena.agents._parsing import DebateDeps
from options_arena.models import AgentResponse  # or TradeThesis for risk

agent: Agent[DebateDeps, AgentResponse] = Agent(
    model=None,
    deps_type=DebateDeps,
    output_type=AgentResponse,
    retries=2,
    model_settings=ModelSettings(extra_body={"num_ctx": 8192}),
)
```

### Prompt Focus Areas (FR-D6)

- **Bull**: upside catalysts, favorable Greeks (high delta, low theta decay), momentum indicators, sector strength
- **Bear**: downside risks, unfavorable Greeks, overbought signals (RSI > 70), macro headwinds, earnings risk
- **Risk**: weigh both cases, identify which is better-supported by data, recommend action (buy/sell/hold), position sizing

### Bear Dynamic Prompt

```python
@bear_agent.system_prompt(dynamic=True)
async def bear_dynamic_prompt(ctx: RunContext[DebateDeps]) -> str:
    base = BEAR_SYSTEM_PROMPT
    if ctx.deps.opponent_argument:
        base += f"\n\n<<<BULL_ARGUMENT>>>\n{ctx.deps.opponent_argument}\n<<<END_BULL_ARGUMENT>>>"
    return base
```

### Output Validator (all three agents)

```python
@agent.output_validator
async def reject_think_tags(ctx: RunContext[DebateDeps], output: OutputType) -> OutputType:
    # Check argument field (AgentResponse) or summary field (TradeThesis)
    if "<think>" in text or "</think>" in text:
        raise ModelRetry("Remove <think> tags from your response")
    return output
```
