# Task: Refactor build_debate_model() into multi-provider dispatcher

## Description

Refactor `build_debate_model()` in `agents/model_config.py` from a Groq-only builder into a multi-provider dispatcher. Extract existing Groq logic into `_build_groq_model()`, add `_build_anthropic_model()` and `_resolve_anthropic_api_key()`. The public API (`build_debate_model(config) -> Model`) stays identical — call sites in `orchestrator.py` are unchanged.

- [ ] `_resolve_anthropic_api_key()` resolves: `config.anthropic_api_key > ANTHROPIC_API_KEY env > None`

- [ ] `mypy --strict` clean (all functions fully annotated)
