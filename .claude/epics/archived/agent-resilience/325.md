# Task: Remove enrichment gate and enhance prompts

## Description

Remove the `and has_enrichment` condition from Flow and Fundamental agent spawning in `orchestrator.py`, add missing-data guidance to both agent system prompts, and add a data-availability note to the context block in `_parsing.py`.

## Acceptance Criteria

- [ ] `orchestrator.py` line 1086: `if flow_output is None and has_enrichment:` changed to `if flow_output is None:`
- [ ] `orchestrator.py` line 1105: `if fundamental_output is None and has_enrichment:` changed to `if fundamental_output is None:`
- [ ] `has_enrichment` variable kept but used only for logging (add `logger.info` when running without enrichment)
- [ ] `flow_agent.py` system prompt includes missing-data instruction about focusing on scan-derived data
- [ ] `fundamental_agent.py` system prompt includes missing-data instruction about focusing on earnings/IV/dividends
- [ ] `_parsing.py` `render_context_block()` appends data-availability note when `enrichment_ratio() == 0.0`
- [ ] All existing tests still pass
- [ ] `ruff check` and `mypy --strict` pass

## Technical Details

### orchestrator.py (lines ~1080-1105)
- Remove `and has_enrichment` from two conditionals
- Keep `has_enrichment = context.enrichment_ratio() > 0.0` for observability
- Add: `if not has_enrichment: logger.info("Running flow/fundamental agents without enrichment data for %s", context.ticker)`

### flow_agent.py (system prompt)
- Append 1-2 sentences after the main instruction block: "If the Unusual Options Flow section is absent from the context block, focus your analysis on put/call ratio, OI concentration from listed contracts, and volume indicators."

### fundamental_agent.py (system prompt)
- Append 1-2 sentences: "If the Fundamental Profile section is absent from the context block, focus on earnings calendar proximity, dividend impact, and IV crush risk assessment."

### _parsing.py (render_context_block)
- After building the context block, check `context.enrichment_ratio() == 0.0`
- If true, append: `"\n\nNote: Enrichment data not available for this ticker. Analysis based on scan-derived indicators."`
