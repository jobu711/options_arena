### Migration (FR-D9)

### Repository methods (FR-D9)

- `save_debate()`: serialize bull/bear JSON via `model_dump_json()`, thesis via `model_dump_json()`, store token count, model name, duration, fallback flag. Return row ID.
- `get_debates_for_ticker()`: query by ticker, ORDER BY created_at DESC, LIMIT N. Return `list[DebateRow]`.
- `DebateRow`: simple `@dataclass` in `repository.py` (not `models/`) — `id`, `scan_run_id`, `ticker`, `bull`, `bear`, `thesis`, `total_tokens`, `model_name`, `duration_ms`, `is_fallback`, `created_at`.
