# Task: Historical close for expired contracts + DTE clamping

## Description

Fix the expired contract P&L calculation in `outcome_collector.py` to use the historical
stock close on expiration day instead of the current quote (FR-1, HIGH priority). Also
add `max(0, ...)` DTE clamping for active contracts (FR-10).

## Acceptance Criteria

- [ ] `_process_expired_contract()` uses `fetch_ohlcv(ticker, period="5d")` to get historical close
- [ ] Filters OHLCV bars for the bar on or before `contract.expiration` date
- [ ] Falls back to `fetch_quote()` with WARNING log if OHLCV data unavailable
- [ ] Weekend/holiday expirations use the last trading day before expiration
- [ ] Active contract DTE uses `max(0, ...)` to prevent negative values
- [ ] Existing tests updated, 2 new tests added

## Technical Details

### Expired contract fix in `_process_expired_contract()`
```python
# Replace fetch_quote() with:
bars = await self._market_data.fetch_ohlcv(ticker, period="5d")
# Filter for bar on or before expiration date
valid_bars = [b for b in bars if b.date <= contract.expiration]
if valid_bars:
    exit_bar = valid_bars[-1]  # most recent bar on or before expiration
    exit_stock_price = exit_bar.close  # Decimal
else:
    logger.warning("No historical bar for %s on %s, falling back to quote", ticker, contract.expiration)
    quote = await self._market_data.fetch_quote(ticker)
    exit_stock_price = quote.price
```

### DTE clamping
