# Task: Composite floor value + explicit risk weight

## Description

Change `_FLOOR_VALUE` from 1.0 to 0.5 in `scoring/composite.py` (FR-5) so bottom-ranked
tickers contribute negative signal via `ln(0.5) = -0.693` instead of zero. Also add
explicit `"risk": 0.0` to `AGENT_VOTE_WEIGHTS` in `agents/orchestrator.py` (FR-9) with
explanatory comment.

- [ ] `_FLOOR_VALUE` changed from `1.0` to `0.5` in `scoring/composite.py`

- [ ] ~4 existing composite tests updated for new floor expectations
- [ ] All existing tests pass with adjusted expectations
