# Task: Word-boundary citation matching

## Description

Replace substring matching with word-boundary regex in `compute_citation_density()` in
`agents/_parsing.py` (FR-8). Currently `label in combined` produces false positives
(e.g., "rsi" matches "surprised"). Use `re.search(r'\b' + re.escape(label) + r'\b', combined)`.

## Acceptance Criteria

- [ ] Citation matching uses `re.search()` with `\b` word boundaries
- [ ] `re.escape()` applied to label to handle special characters
- [ ] False positives eliminated (e.g., "rsi" no longer matches "surprised")
- [ ] True matches still detected (e.g., "RSI" matches "RSI is at 72")
- [ ] 3 new tests pass

import re
