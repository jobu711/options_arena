# Task: NEUTRAL exclusion from agreement denominator

## Description

Fix `compute_agreement_score()` in `agents/orchestrator.py` to exclude NEUTRAL agents
from the denominator (FR-3). Currently NEUTRAL agents dilute agreement score while
bull/bear scores exclude them, creating inconsistent results.

## Acceptance Criteria

- [ ] Denominator changed from `len(agent_directions)` to `bullish_count + bearish_count`
- [ ] All-NEUTRAL agents produce agreement = 0.0 (no directional consensus)
- [ ] Existing agreement tests updated for new denominator
- [ ] ~5 existing tests updated to reflect NEUTRAL exclusion

## Technical Details

### Change in `agents/orchestrator.py`
```python
def compute_agreement_score(agent_directions: ...) -> float:
    # Count directional agents only
    directional_count = bullish_count + bearish_count
    if directional_count == 0:
        return 0.0  # All NEUTRAL — no directional consensus
    majority = max(bullish_count, bearish_count)
    return majority / directional_count
```
