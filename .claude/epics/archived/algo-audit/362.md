# Task: Single-ticker normalization for ad-hoc debate

## Description

### Normalization function
```python
def normalize_single_ticker(signals: IndicatorSignals) -> IndicatorSignals:
    """Normalize raw indicator signals to 0-100 via domain-bound linear scaling."""
    normalized = {}
    for field, (lo, hi) in DOMAIN_BOUNDS.items():
        value = getattr(signals, field, None)
        if value is not None and math.isfinite(value):
            normalized[field] = max(0.0, min(100.0, (value - lo) / (hi - lo) * 100))
        else:
            normalized[field] = value
    result = signals.model_copy(update=normalized)
    return invert_indicators(result)
```

### Edge Cases
- NaN indicator value → preserved as-is (no crash)
- All indicators at None → passthrough, no crash
- Extreme outlier (RSI=200) → clamped to 100.0
- Zero-width domain (shouldn't happen with these bounds)
