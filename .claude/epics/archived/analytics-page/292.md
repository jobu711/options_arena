### SVG Layout
```
Score Calibration        [Bucket: 10 ▼]

 Avg Return %           Win Rate %
  +20% ┌──┐                         100%
       │  │  ┌──┐  ●───●
  +10% │  │  │  │ /     \───●        60%
       │  │──│  │/          │
    0% ├──┤──├──┤──┌──┐     │        40%
       │  │  │  │  │  │  ┌──┐
  -10% └──┘  └──┘  └──┘  └──┘        0%
       0-10  10-20 20-30  ...  90-100
        n=5   n=12  n=8   ...   n=3
```

### Coordinate Math
- viewBox: `0 0 600 300`
- Bar series: computed from `avg_return_pct`, can be positive or negative (bars go up/down from zero line)
- Line series: computed from `win_rate` (0-100 scale)
- Zero line for returns at mid-height
- Each bucket gets equal horizontal space: `chartWidth / buckets.length`
