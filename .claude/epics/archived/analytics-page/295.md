### Formatting Rules
- Win rate, avg returns: `value !== null ? (value * 1).toFixed(1) + '%' : '--'` (backend returns as percentage already)
- Verify: check if backend returns 63.2 or 0.632 — format accordingly
- Total contracts: `Intl.NumberFormat` with thousands separator
- Best direction: capitalize first letter
- Best holding period: `N days` or `"--"`
