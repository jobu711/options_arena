### SVG Layout
```
Delta Performance        [Holding: 10 days ▼]

  +30% ┌─┐┌─┐
       │R││W│ ┌─┐┌─┐
  +20% │ ││ │ │R││W│
       │ ││ │ │ ││ │ ┌─┐┌─┐
  +10% │ ││ │ │ ││ │ │R││W│
    0% └─┘└─┘ └─┘└─┘ └─┘└─┘
       0.2-0.3 0.3-0.4 0.4-0.5 ...

  R = Avg Return   W = Win Rate
```

### Coordinate Math
- viewBox: `0 0 600 300`
- Groups: one per delta bucket
- 2 bars per group: return (left) + win_rate (right)
- Bar width: `groupWidth / 2 - gap`
- Group width: `chartWidth / buckets.length`
- Return bars can be negative (below zero line)
- Win rate bars always positive (0-100% scale)
