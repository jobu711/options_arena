# Task: Foundation — Models, Config, Migrations

## Description

Create all Pydantic models for the analytics persistence layer, the `OutcomeCollectionMethod` enum, `AnalyticsConfig` sub-config, and 3 SQL migration files. This is the foundation task — every subsequent task depends on these data shapes existing.

All models go in a single new file `src/options_arena/models/analytics.py`. All are frozen snapshot models with `isfinite()` validators on floats, `is_finite()` on Decimals, UTC datetime validators, and Decimal field serializers.

## Technical Details

### New File: `src/options_arena/models/analytics.py`

9 frozen models:

1. **`RecommendedContract`** — mirrors `recommended_contracts` table schema
   - `id: int | None = None` (DB-assigned)
   - `scan_run_id: int`
   - `ticker: str`
   - `option_type: OptionType`
   - `strike: Decimal`, `bid: Decimal`, `ask: Decimal`, `last: Decimal | None`
   - `expiration: date`
   - `volume: int`, `open_interest: int`
   - `market_iv: float`
   - `exercise_style: ExerciseStyle`
   - `delta: float | None`, `gamma: float | None`, `theta: float | None`, `vega: float | None`, `rho: float | None`
   - `pricing_model: PricingModel | None`
   - `greeks_source: GreeksSource | None`
   - `entry_stock_price: Decimal`, `entry_mid: Decimal`
   - `direction: SignalDirection`
   - `composite_score: float`
   - `risk_free_rate: float`
   - `created_at: datetime`
   - Validators: `isfinite()` on `market_iv`, `composite_score`, `risk_free_rate`; `is_finite()` on all Decimal fields; UTC on `created_at`; delta in [-1,1] if not None; gamma/vega >= 0 if not None
   - `field_serializer` for all Decimal fields
   - `computed_field` for `mid -> Decimal` = `(bid + ask) / Decimal("2")`

2. **`ContractOutcome`** — mirrors `contract_outcomes` table schema
   - `id: int | None = None`
   - `recommended_contract_id: int`
   - `exit_stock_price: Decimal | None`
   - `exit_contract_mid: Decimal | None`, `exit_contract_bid: Decimal | None`, `exit_contract_ask: Decimal | None`
   - `exit_date: date | None`
   - `stock_return_pct: float | None`, `contract_return_pct: float | None`
   - `is_winner: bool | None`
   - `holding_days: int | None`
   - `dte_at_exit: int | None`
   - `collection_method: OutcomeCollectionMethod`
   - `collected_at: datetime`
   - Validators: UTC on `collected_at`; `isfinite()` on return pcts if not None; `is_finite()` on Decimals if not None

3. **`NormalizationStats`** — mirrors `normalization_metadata` table
   - `id: int | None = None`
   - `scan_run_id: int`
   - `indicator_name: str`
   - `ticker_count: int`
   - `min_value: float | None`, `max_value: float | None`
   - `median_value: float | None`, `mean_value: float | None`, `std_dev: float | None`
   - `p25: float | None`, `p75: float | None`
   - `created_at: datetime`
   - Validators: UTC on `created_at`; `isfinite()` on all float fields if not None; `ticker_count >= 0`

4. **`WinRateResult`** — analytics query result
   - `direction: SignalDirection`
   - `total_contracts: int`
   - `winners: int`
   - `losers: int`
   - `win_rate: float`
   - Validator: `isfinite()` on `win_rate`; 0.0 <= win_rate <= 1.0

5. **`ScoreCalibrationBucket`** — analytics query result
   - `score_min: float`, `score_max: float`
   - `contract_count: int`
   - `avg_return_pct: float`
   - `win_rate: float`
   - Validators: `isfinite()` on all floats; 0.0 <= win_rate <= 1.0

6. **`IndicatorAttributionResult`** — analytics query result
   - `indicator_name: str`
   - `holding_days: int`
   - `correlation: float`
   - `avg_return_when_high: float`
   - `avg_return_when_low: float`
   - `sample_size: int`
   - Validators: `isfinite()` on correlation, avg_returns; -1.0 <= correlation <= 1.0

7. **`HoldingPeriodResult`** — analytics query result
   - `holding_days: int`
   - `direction: SignalDirection`
   - `avg_return_pct: float`
   - `median_return_pct: float`
   - `win_rate: float`
   - `sample_size: int`
   - Validators: `isfinite()` on all floats; 0.0 <= win_rate <= 1.0

8. **`DeltaPerformanceResult`** — analytics query result
   - `delta_min: float`, `delta_max: float`
   - `holding_days: int`
   - `avg_return_pct: float`
   - `win_rate: float`
   - `sample_size: int`
   - Validators: `isfinite()` on all floats; 0.0 <= win_rate <= 1.0

9. **`PerformanceSummary`** — aggregate analytics result
   - `lookback_days: int`
   - `total_contracts: int`
   - `total_with_outcomes: int`
   - `overall_win_rate: float | None`
   - `avg_stock_return_pct: float | None`
   - `avg_contract_return_pct: float | None`
   - `best_direction: SignalDirection | None`
   - `best_holding_days: int | None`
   - Validators: `isfinite()` on floats if not None; win_rate in [0,1] if not None

### New File: `data/migrations/011_recommended_contracts.sql`

```sql
CREATE TABLE IF NOT EXISTS recommended_contracts (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    scan_run_id INTEGER NOT NULL REFERENCES scan_runs(id),
    ticker TEXT NOT NULL,
    option_type TEXT NOT NULL,
    strike TEXT NOT NULL,
    expiration TEXT NOT NULL,
    bid TEXT NOT NULL,
    ask TEXT NOT NULL,
    last TEXT,
    volume INTEGER NOT NULL,
    open_interest INTEGER NOT NULL,
    market_iv REAL NOT NULL,
    exercise_style TEXT NOT NULL,
    delta REAL, gamma REAL, theta REAL, vega REAL, rho REAL,
    pricing_model TEXT,
    greeks_source TEXT,
    entry_stock_price TEXT NOT NULL,
    entry_mid TEXT NOT NULL,
    direction TEXT NOT NULL,
    composite_score REAL NOT NULL,
    risk_free_rate REAL NOT NULL,
    created_at TEXT NOT NULL,
    UNIQUE(scan_run_id, ticker, option_type, strike, expiration)
);
CREATE INDEX IF NOT EXISTS idx_rc_scan_run ON recommended_contracts(scan_run_id);
CREATE INDEX IF NOT EXISTS idx_rc_ticker ON recommended_contracts(ticker);
CREATE INDEX IF NOT EXISTS idx_rc_expiration ON recommended_contracts(expiration);
```

### New File: `data/migrations/012_contract_outcomes.sql`

```sql
CREATE TABLE IF NOT EXISTS contract_outcomes (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    recommended_contract_id INTEGER NOT NULL REFERENCES recommended_contracts(id),
    exit_stock_price TEXT,
    exit_contract_mid TEXT,
    exit_contract_bid TEXT,
    exit_contract_ask TEXT,
    exit_date TEXT,
    stock_return_pct REAL,
    contract_return_pct REAL,
    is_winner INTEGER,
    holding_days INTEGER,
    dte_at_exit INTEGER,
    collection_method TEXT NOT NULL,
    collected_at TEXT NOT NULL,
    UNIQUE(recommended_contract_id, exit_date)
);
CREATE INDEX IF NOT EXISTS idx_co_rec_id ON contract_outcomes(recommended_contract_id);
```

### New File: `data/migrations/013_normalization_metadata.sql`

```sql
CREATE TABLE IF NOT EXISTS normalization_metadata (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    scan_run_id INTEGER NOT NULL REFERENCES scan_runs(id),
    indicator_name TEXT NOT NULL,
    ticker_count INTEGER NOT NULL,
    min_value REAL, max_value REAL,
    median_value REAL, mean_value REAL, std_dev REAL,
    p25 REAL, p75 REAL,
    created_at TEXT NOT NULL,
    UNIQUE(scan_run_id, indicator_name)
);
CREATE INDEX IF NOT EXISTS idx_nm_scan_run ON normalization_metadata(scan_run_id);
```

### Edge Cases
- NaN/Inf values: rejected by `isfinite()` validators on all float fields
- Non-UTC datetime: rejected by UTC validator
- Decimal precision loss: `Decimal("1.05")` must survive JSON roundtrip
- Empty/None Greek fields: all 5 Greek fields are optional
- Zero ticker_count: should be accepted (valid empty scan)

### Mocking Strategy
- No mocking needed — pure model construction and validation tests
- Use `pytest.raises(ValidationError)` for rejection tests
- Use `monkeypatch` for env var override tests on `AppSettings`
