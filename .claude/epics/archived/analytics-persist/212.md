# Task: Outcome Tracking Service and CLI

## Description

Create the `OutcomeCollector` service that queries persisted contracts, fetches current market prices, computes P&L, and persists outcomes. Add repository methods for outcome CRUD. Add CLI `outcomes` subcommand group (`collect`, `summary`). Handle expired contracts via intrinsic value computation.

## Acceptance Criteria

- [ ] `OutcomeCollector` class created in `services/outcome_collector.py`
- [ ] `collect_outcomes(holding_days)` method queries contracts from N days ago without outcomes at that period
- [ ] Stock return computed as `(exit_price - entry_price) / entry_price * 100`
- [ ] Contract return computed from mid prices (or intrinsic value for expired)
- [ ] Expired contracts handled: intrinsic value `max(0, S-K)` for calls, `max(0, K-S)` for puts
- [ ] Worthless expired contracts: return = -100%, `collection_method = expired_worthless`

### New File: `src/options_arena/services/outcome_collector.py`

```python
class OutcomeCollector:
    def __init__(
        self,
        config: AnalyticsConfig,
        repository: Repository,
        market_data: MarketDataService,
    ) -> None:
        """DI constructor — config, repository, and market_data injected."""

    async def collect_outcomes(
        self,
        holding_days: int | None = None,
    ) -> list[ContractOutcome]:
        """Collect outcomes for contracts from holding_days ago.

        If holding_days is None, iterates all configured holding periods.
        Never raises — logs errors and returns partial results.
        """

    async def _collect_for_period(
        self,
        holding_days: int,
    ) -> list[ContractOutcome]:
        """Collect outcomes for a single holding period."""

    def _compute_stock_return(
        self,
        entry_price: Decimal,
        exit_price: Decimal,
    ) -> float:
        """Compute stock return percentage."""

    def _compute_contract_return(
        self,
        entry_mid: Decimal,
        exit_mid: Decimal,
    ) -> float:
        """Compute contract return percentage from mid prices."""

    def _compute_intrinsic_value(
        self,
        option_type: OptionType,
        strike: Decimal,
        stock_price: Decimal,
    ) -> Decimal:
        """Compute intrinsic value for expired contracts.

        Call: max(0, S - K)
        Put:  max(0, K - S)
        """

    def _is_expired(self, expiration: date) -> bool:
        """Check if contract has expired."""

    async def get_summary(
        self,
        lookback_days: int = 30,
    ) -> PerformanceSummary:
        """Get aggregate performance summary."""
```

**Collection flow:**
1. Query `get_contracts_needing_outcomes(holding_days, lookback_date)`
   - `lookback_date = today - holding_days` (contracts created on that date)
2. For each contract:
   a. If expired: compute intrinsic value from current stock price
      - If intrinsic > 0: `collection_method = intrinsic`, return based on intrinsic
      - If intrinsic == 0: `collection_method = expired_worthless`, return = -100%
   b. If still active: fetch current stock quote + try to get option mid
      - `collection_method = market`
3. Persist all outcomes via `save_contract_outcomes()`
4. Return collected outcomes

**Error handling per contract:**
- `fetch_quote()` fails → skip contract, log WARNING
- Option chain unavailable for active contract → compute return from stock price only, `contract_return_pct = None`
- Any unexpected exception → skip contract, log ERROR

### `data/repository.py` Changes

Add 4 methods under `# --- Analytics: Outcomes ---`:

1. `save_contract_outcomes(outcomes: list[ContractOutcome]) -> None`
   - `executemany()` for batch insert
   - Store Decimal as TEXT
   - `await db.commit()`

2. `get_outcomes_for_contract(contract_id: int) -> list[ContractOutcome]`
   - SELECT WHERE recommended_contract_id = ? ORDER BY holding_days
   - Use `_row_to_contract_outcome()` converter

3. `get_contracts_needing_outcomes(holding_days: int, lookback_date: date) -> list[RecommendedContract]`
   - SELECT from recommended_contracts WHERE:
     - `date(created_at) = ?` (contracts from the lookback date)
     - NOT EXISTS (SELECT 1 FROM contract_outcomes WHERE recommended_contract_id = rc.id AND holding_days = ?)
   - Returns contracts that don't yet have outcomes for this holding period

4. `has_outcome(contract_id: int, exit_date: date) -> bool`
   - SELECT EXISTS check for duplicate prevention

### Edge Cases
- Contract with no matching quote available (market closed, delisted ticker)
- Expired contract with zero intrinsic value (OTM at expiry)
- Contract expiring today (edge case for is_expired check)
- Division by zero: entry_mid == 0 (should not happen with valid data, but guard)
- All contracts fail to collect: return empty list, no error
- Multiple holding periods in single collect call
