## Technical Details

### `outcomes backtest` command
```python
@outcomes_app.command()
def backtest(
    holding_days: Annotated[int, typer.Option()] = 20,
) -> None:
    """Show backtesting performance summary."""
    asyncio.run(_backtest_async(holding_days))
```
Fetches: `get_holding_period_comparison()`, `get_equity_curve()`, `get_drawdown_series()`.
Renders Rich table with columns: Holding Period, Direction, Trades, Win Rate, Avg Return, Median Return, Sharpe, Max Loss.

### `outcomes equity-curve` command
```python
@outcomes_app.command(name="equity-curve")
def equity_curve(
    direction: Annotated[str | None, typer.Option()] = None,
    period: Annotated[int | None, typer.Option()] = None,
) -> None:
    """Show cumulative equity curve."""
    asyncio.run(_equity_curve_async(direction, period))
```
Fetches: `get_equity_curve(direction, period)`.
Renders Rich table with columns: Date, Cumulative Return %, Trades.

### Pattern
Follow existing `outcomes summary` and `outcomes collect` patterns in `cli/outcomes.py`. Sync command → `asyncio.run()`. Service created with config, used in try/finally.
