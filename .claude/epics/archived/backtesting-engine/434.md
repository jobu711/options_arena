## Technical Details

### New Endpoints

```python
router = APIRouter(prefix="/api/analytics/backtest", tags=["backtest"])

@router.get("/equity-curve")
async def get_equity_curve(
    direction: str | None = None,
    period: int | None = None,
    repo: Repository = Depends(get_repo),
) -> list[EquityCurvePoint]: ...

@router.get("/drawdown")
async def get_drawdown(
    period: int | None = None,
    repo: Repository = Depends(get_repo),
) -> list[DrawdownPoint]: ...

@router.get("/sector-performance")
async def get_sector_performance(
    holding_days: int = 20,
    repo: Repository = Depends(get_repo),
) -> list[SectorPerformanceResult]: ...

@router.get("/dte-performance")
async def get_dte_performance(
    holding_days: int = 20,
    repo: Repository = Depends(get_repo),
) -> list[DTEBucketResult]: ...

@router.get("/iv-performance")
async def get_iv_performance(
    holding_days: int = 20,
    repo: Repository = Depends(get_repo),
) -> list[IVRankBucketResult]: ...

@router.get("/greeks-decomposition")
async def get_greeks_decomposition(
    groupby: str = "direction",
    holding_days: int = 20,
    repo: Repository = Depends(get_repo),
) -> list[GreeksDecompositionResult]: ...

@router.get("/holding-comparison")
async def get_holding_comparison(
    repo: Repository = Depends(get_repo),
) -> list[HoldingPeriodComparison]: ...
```

### Router Registration (`app.py`)
```python
from options_arena.api.routes.backtest import router as backtest_router
app.include_router(backtest_router)
```

### Pattern
Follow existing `routes/analytics.py` endpoint pattern. Thin layer: validate params → call repo method → return model. FastAPI handles serialization.

### Edge Cases
- No data in database: 200 with empty list `[]`
- Invalid direction parameter: 422 validation error
- Negative holding_days: 422 validation error
- Very large period value: works but returns all data

### Mocking Strategy
- Use `httpx.AsyncClient` with test app factory
- Seed in-memory SQLite with test data
- No external API mocking needed (all data from local DB)
