# Task: Agent prediction persistence

## Description

Implement FR-8 (agent prediction persistence). Create the `agent_predictions` SQLite table via migration, add the `AgentPrediction` model, add `Repository.save_agent_predictions()`, and wire into the 3 persistence sites: `_persist_result()` (orchestrator), API single-debate, and API batch-debate.

This enables per-agent accuracy tracking for future self-calibrating weights.

## Acceptance Criteria

- [ ] Migration `025_agent_predictions.sql` creates table with FK to `ai_theses`
- [ ] `AgentPrediction` Pydantic model: frozen, UTC validator on `created_at`, confidence validator [0.0, 1.0]
- [ ] `Repository.save_agent_predictions(debate_id: int, result: DebateResult) -> None` method
- [ ] Orchestrator `_persist_result()`: captures `save_debate()` return value as `debate_id`, calls `save_agent_predictions(debate_id, result)`
- [ ] API single-debate route: captures `debate_id`, calls `save_agent_predictions()`
- [ ] API batch-debate route: adds `save_agent_predictions()` after existing `debate_id` capture
- [ ] Parameterized SQL (no string interpolation)
- [ ] `UNIQUE(debate_id, agent_name)` constraint prevents duplicate predictions
- [ ] `ruff check` + `ruff format` clean
- [ ] `mypy --strict` passes

### Implementation Approach

**Migration (`data/migrations/025_agent_predictions.sql`):**
```sql
CREATE TABLE IF NOT EXISTS agent_predictions (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    debate_id INTEGER NOT NULL REFERENCES ai_theses(id),
    recommended_contract_id INTEGER REFERENCES recommended_contracts(id),
    agent_name TEXT NOT NULL,
    direction TEXT,
    confidence REAL NOT NULL,
    created_at TEXT NOT NULL,
    UNIQUE(debate_id, agent_name)
);
CREATE INDEX IF NOT EXISTS idx_ap_debate ON agent_predictions(debate_id);
CREATE INDEX IF NOT EXISTS idx_ap_contract ON agent_predictions(recommended_contract_id);
```

**Model (`models/analysis.py`):**
- `AgentPrediction(BaseModel)`: frozen, fields: `debate_id: int`, `agent_name: str`, `direction: SignalDirection | None`, `confidence: float`, `created_at: datetime`
- UTC validator on `created_at`, confidence validator [0.0, 1.0], `isfinite()` on confidence

**Repository (`data/repository.py`):**
- `save_agent_predictions(debate_id: int, result: DebateResult) -> None`
- Extract agent name, direction, confidence from each agent response field on `DebateResult`
- Iterate: `bull_response`, `bear_response`, `flow_response`, `fundamental_response`, `trend_response`, `volatility_response`
- Use `INSERT OR IGNORE` with `UNIQUE(debate_id, agent_name)` for idempotency
- Parameterized SQL only

**Orchestrator Wiring:**
- In `_persist_result()`: capture `debate_id = await repository.save_debate(...)`, then `await repository.save_agent_predictions(debate_id, result)`
- Change return type to `int | None`

**API Wiring:**
- Single-debate: capture `debate_id` from `repo.save_debate()`, call `save_agent_predictions()`
- Batch-debate: already captures `debate_id` — add `save_agent_predictions()` after

### Key Considerations

- Empty DebateResult (all agents failed): no predictions saved, no error
