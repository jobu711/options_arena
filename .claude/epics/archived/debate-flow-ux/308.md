# Task: Create ConsensusPanel component and integrate in DebateResultPage with back-to-scan nav

## Description
Create a new `ConsensusPanel.vue` component that renders v2 consensus data (agreement score,
dissenting agents, contrarian challenge text) and integrate it into `DebateResultPage.vue`.
Also add a "Back to Scan Results" navigation button visible when the debate was launched from
a scan. The component is only shown for v2 debates where `agent_agreement_score` is non-null.

## Acceptance Criteria
- [ ] `ConsensusPanel.vue` component created in `web/src/components/`
- [ ] Component renders agreement percentage as a visual bar
- [ ] Component lists dissenting agent names
- [ ] Component shows contrarian challenge text
- [ ] Component is hidden when `agreementScore` is null (v1 debates)
- [ ] `DebateResultPage.vue` imports and places ConsensusPanel between thesis banner and agent cards
- [ ] `data-testid="consensus-panel"` attribute present for E2E testing
- [ ] "Back to Scan Results" button visible when `scan_run_id` is set
- [ ] "Back to Scan Results" button navigates to `/scan/{scan_run_id}`
- [ ] "Back to Scan Results" button hidden for standalone debates (no `scan_run_id`)

## Technical Details

### ConsensusPanel.vue (new file ~100-150 lines)

**Location**: `web/src/components/ConsensusPanel.vue`

**Props**:
```typescript
interface Props {
  agreementScore: number | null
  agentsCompleted: number | null
  dissentingAgents: string[]
  contrarianDissent: string | null
}
```

**Rendering**:
- Agreement score as percentage bar (PrimeVue `ProgressBar` or custom)
- `agentsCompleted` shown as "X/8 agents participated" context
- Dissenting agent names as PrimeVue `Tag` pills
- Contrarian challenge as a styled blockquote or card section
- Hidden entirely when `agreementScore` is null (via `v-if` on parent)

**Styling**:
- Use scoped CSS with project accent colors from `variables.css`
- Monospace font for numeric values (`var(--font-mono)`)
- Match existing dark theme Aura card styling

### DebateResultPage.vue changes (~30 lines)

- Import `ConsensusPanel` component
- Place between thesis banner section and agent cards grid
- Conditional rendering: `v-if="isV2 && debate?.agent_agreement_score != null"`
- Pass props from `debate` store state
- Add `data-testid="consensus-panel"` on wrapper

### Back-to-scan navigation (~10 lines)

- Add PrimeVue `Button` with `severity="secondary"`, `icon="pi pi-arrow-left"`
- Label: "Back to Scan Results"
- `v-if="debate?.scan_run_id"` — only shown when debate was launched from scan
- `@click="router.push('/scan/' + debate.scan_run_id)"`
- Place in page header area near existing controls
