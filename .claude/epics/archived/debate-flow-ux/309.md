# Task: Wire debate button in TickerDrawer with modal, WebSocket, and navigation

## Description

to the full debate flow: POST → DebateProgressModal → WebSocket streaming → auto-navigate
to `/debate/{id}` on completion. Replicate the exact pattern from `ScanResultsPage.vue`

**Add local state**:
```typescript
const debateStore = useDebateStore()
const operationStore = useOperationStore()
const toast = useToast()
const debateModalVisible = ref(false)
const debatingTicker = ref('')
const isDebating = ref(false)
```

**Add click handler**:
```typescript
async function startDebate() {
  if (!props.score || isDebating.value) return
  const ticker = props.score.ticker
  debatingTicker.value = ticker
  debateModalVisible.value = true
  isDebating.value = true
  try {
    await debateStore.startDebate(ticker, props.scanId)
    // DebateProgressModal handles WebSocket + navigation
  } catch (e) {
    toast.add({ severity: 'error', summary: 'Debate failed', detail: String(e), life: 5000 })
  } finally {
    isDebating.value = false
  }
}
```

**Button template update**:
```vue
<Button
  label="Debate This Ticker"
  icon="pi pi-comments"
  :disabled="operationStore.inProgress || isDebating"
  v-tooltip="operationStore.inProgress ? 'Operation in progress' : undefined"
  @click="startDebate"
/>
```

**Add modal template**:
```vue
<DebateProgressModal
  v-model:visible="debateModalVisible"
  :ticker="debatingTicker"
  :agents="debateStore.agentProgress"
  :error="debateStore.error"
/>
```

- WebSocket disconnects mid-debate — modal shows reconnection state
- User closes drawer while debate in progress — cleanup via `onUnmounted`
- `score` prop is null — `startDebate()` early returns
