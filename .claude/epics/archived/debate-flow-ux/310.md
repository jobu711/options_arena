- Debate fails mid-flow — modal shows error, button re-enables
- WebSocket disconnects — modal shows reconnection state
- Database has v1 debate without consensus fields — panel hidden
- `scan_run_id` is 0 — treated as falsy? (should be truthy since it's a valid ID)
