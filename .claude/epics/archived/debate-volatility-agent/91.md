# Task: Implement Volatility Agent

## Description
Create the Volatility Agent in `agents/volatility.py`. This is a new PydanticAI agent that assesses whether implied volatility is mispriced and recommends vol-specific strategies. It uses a `dynamic=True` system prompt that injects bull + bear arguments when available, uses `PROMPT_RULES_APPENDIX` from `_parsing.py`, and has an output validator that strips `<think>` tags via the shared `build_cleaned_volatility_thesis()` helper.

## Acceptance Criteria
- [ ] `agents/volatility.py` created (~100 lines)
- [ ] Module-level `volatility_agent: Agent[DebateDeps, VolatilityThesis]` instance
- [ ] `model=None` at init, `deps_type=DebateDeps`, `output_type=VolatilityThesis`, `retries=2`
- [ ] `ModelSettings(extra_body={"num_ctx": 8192})` matching other agents
- [ ] `dynamic=True` system prompt that injects bull + bear arguments via `<<<BULL_ARGUMENT>>>` and `<<<BEAR_ARGUMENT>>>` delimiters when available in `ctx.deps`
- [ ] System prompt focuses on: IV Rank, IV Percentile, ATM IV 30D, historical vol context
- [ ] Prompt includes `PROMPT_RULES_APPENDIX` from `_parsing.py`
- [ ] Version header: `# VERSION: v1.0` above prompt constant
- [ ] `@volatility_agent.output_validator` that delegates to `build_cleaned_volatility_thesis()`
- [ ] Re-exported from `agents/__init__.py`
- [ ] No inter-agent imports (only imports from `_parsing.py` and `models/`)
- [ ] Uses `logging.getLogger(__name__)`, no `print()`
