# Task: Context rendering + Fundamental agent prompt v3.0

## Description
Add 3 new FD context sections to `render_fundamental_context()` and `render_context_block()` in `agents/_parsing.py`. Bump the fundamental agent system prompt to v3.0 with financial health, valuation depth, and growth trajectory analysis requirements.

## Acceptance Criteria
- [ ] `render_fundamental_context()` renders 3 new sections when fd_* fields present:
  1. "## Income Statement (TTM)" — fd_revenue, fd_net_income, fd_operating_income, fd_eps_diluted, fd_gross_margin, fd_operating_margin, fd_net_margin
  2. "## Balance Sheet" — fd_total_debt, fd_total_cash, fd_total_assets, fd_current_ratio
  3. "## Growth & Valuation" — fd_revenue_growth, fd_earnings_growth, fd_ev_to_ebitda, fd_free_cash_flow_yield
- [ ] `render_context_block()` also renders the same 3 sections
- [ ] Sections only rendered when at least one field in that section is non-None
- [ ] Each field None-checked individually before rendering its line
- [ ] Formatting: dollar amounts with `$` prefix and abbreviation (B/M), percentages with `%`, ratios as-is
- [ ] `FUNDAMENTAL_SYSTEM_PROMPT` bumped to v3.0
- [ ] Prompt adds analysis requirements: financial health (D/E + current ratio), valuation depth (EV/EBITDA + FCF yield + PEG), growth trajectory (revenue growth + earnings growth trends)
- [ ] Prompt adds interpretation rules (high D/E + low current ratio = risk, PEG > 2.0 = overvalued, negative FCF yield = cash burn)

## Technical Details
- **Modified files**: `src/options_arena/agents/_parsing.py`, `src/options_arena/agents/fundamental_agent.py`
- Insert new sections in `render_fundamental_context()` after the "Fundamental Profile" section and before "Analyst Intelligence"
- Section rendering pattern (from existing code):
  ```python
  lines: list[str] = []
  if ctx.fd_revenue is not None:
      lines.append(f"Revenue: ${ctx.fd_revenue / 1e9:.1f}B")
  if lines:
      sections.append("## Income Statement (TTM)\n" + "\n".join(lines))
  ```
- For `render_context_block()`: same sections appended to the fundamental block
- Prompt version bump: change `VERSION = "v2.X"` to `VERSION = "v3.0"` in fundamental_agent.py
- New prompt text should be concise — 3-4 bullet points per analysis dimension
