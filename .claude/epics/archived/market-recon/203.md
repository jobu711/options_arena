# Task: Orchestrator wiring — build_market_context + run_debate_v2

## Description
Extend `build_market_context()` in `agents/orchestrator.py` to accept an `intelligence: IntelligencePackage | None` parameter and map all 8 intelligence fields to MarketContext. Map 22 DSE fields from `TickerScore.signals` (10 individual indicators + 3 second-order Greeks) and `TickerScore.dimensional_scores` (8 family sub-scores) + `direction_confidence`. Update `run_debate()` and `run_debate_v2()` signatures to pass intelligence through.

## Acceptance Criteria
- [ ] `build_market_context()` accepts `intelligence: IntelligencePackage | None = None` parameter
- [ ] 8 intelligence fields mapped from IntelligencePackage to MarketContext (analyst_target_mean, analyst_target_upside_pct, analyst_consensus_score, analyst_upgrades_30d, analyst_downgrades_30d, insider_net_buys_90d, insider_buy_ratio, institutional_pct)
- [ ] 8 dimensional scores mapped from `TickerScore.dimensional_scores` (dim_trend ← .trend, etc.)
- [ ] 10 DSE individual indicators mapped from `TickerScore.signals` (vol_regime, iv_hv_spread, gex, unusual_activity_score, skew_ratio, vix_term_structure, market_regime, rsi_divergence, expected_move, expected_move_ratio)
- [ ] 3 second-order Greeks mapped from `TickerScore.signals` (vanna, charm, vomma → target_vanna, target_charm, target_vomma)
- [ ] 1 direction_confidence mapped from `TickerScore.direction_confidence`
- [ ] `run_debate()` and `run_debate_v2()` accept and pass through intelligence parameter
- [ ] All None passthrough when intelligence is None or TickerScore has no dimensional_scores
- [ ] Ruff, mypy, full test suite pass
