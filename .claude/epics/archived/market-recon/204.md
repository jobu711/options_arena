# Task: CLI and API integration wiring

## Description
Wire `IntelligenceService` into the CLI debate command (`--no-recon` flag, conditional service creation/cleanup) and the API layer (lifespan creation on `app.state`, dependency provider, debate route fetching). Follows the exact OpenBB wiring pattern at all 5 DI sites.

## Acceptance Criteria
- [ ] CLI: `--no-recon` flag added to debate command (mirrors `--no-openbb`)
- [ ] CLI: `IntelligenceService` created conditionally when `not no_recon and settings.intelligence.enabled`
- [ ] CLI: Intelligence passed to `run_debate_v2()` / `run_debate()`
- [ ] CLI: `IntelligenceService.close()` called in `finally` block
- [ ] API: `IntelligenceService` created in `lifespan()` when config enabled
- [ ] API: Stored on `app.state.intelligence`
- [ ] API: `close()` called in shutdown
- [ ] API: `get_intelligence()` dependency provider in `deps.py`
- [ ] API: Debate route fetches intelligence before debate call
- [ ] `--no-recon` suppresses intelligence sections; DSE sections remain (DSE is not gated)
- [ ] Ruff, mypy, full test suite pass

## Technical Details

### Files to Modify
- `src/options_arena/cli/commands.py` — `--no-recon` flag, service creation, pass to orchestrator
- `src/options_arena/api/app.py` — Lifespan: create `IntelligenceService`, store on `app.state`, close in shutdown
- `src/options_arena/api/deps.py` — `get_intelligence()` dependency provider
- `src/options_arena/api/routes/debate.py` — Fetch intelligence, pass to orchestrator

### CLI pattern (mirrors --no-openbb)
```python
no_recon: bool = typer.Option(False, "--no-recon", help="Skip intelligence fetching")
# ...
intelligence_svc: IntelligenceService | None = None
if not no_recon and settings.intelligence.enabled:
    intelligence_svc = IntelligenceService(settings.intelligence, cache, limiter)
try:
    intel = await intelligence_svc.fetch_intelligence(ticker, float(quote.price)) if intelligence_svc else None
    result = await run_debate_v2(..., intelligence=intel)
finally:
    if intelligence_svc is not None:
        await intelligence_svc.close()
```

### API pattern (mirrors OpenBB)
```python
# app.py lifespan
intelligence_svc: IntelligenceService | None = None
if settings.intelligence.enabled:
    intelligence_svc = IntelligenceService(settings.intelligence, cache, limiter)
app.state.intelligence = intelligence_svc
# ... shutdown
if intelligence_svc is not None:
    await intelligence_svc.close()

# deps.py
def get_intelligence(request: Request) -> IntelligenceService | None:
    return getattr(request.app.state, "intelligence", None)

# debate route
intelligence_svc = getattr(request.app.state, "intelligence", None)
intel: IntelligencePackage | None = None
if intelligence_svc is not None:
    intel = await intelligence_svc.fetch_intelligence(ticker, float(quote.price))
result = await run_debate_v2(..., intelligence=intel)
```
