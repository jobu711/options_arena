### Ratio methods pattern (parallel to existing completeness_ratio/enrichment_ratio)
```python
def intelligence_ratio(self) -> float:
    """Fraction of 8 intelligence fields populated."""
    fields = [self.analyst_target_mean, self.analyst_target_upside_pct, ...]
    populated = sum(1 for f in fields if f is not None)
    return populated / len(fields)
```
