# Task: Pipeline integration — Phase 1 loading + Phase 3 write-back

## Description
Integrate the metadata index into the scan pipeline at two points: Phase 1 loads cached metadata to extend `sector_map`/`industry_group_map` beyond S&P 500, and Phase 3 writes back metadata for every ticker it processes via `map_yfinance_to_metadata()`.

## Acceptance Criteria
- [ ] Phase 1 (`_phase_universe()`): After S&P 500 map construction, loads `get_all_ticker_metadata()` from repository
- [ ] Phase 1: Merges metadata into `sector_map` and `industry_group_map` (S&P 500 CSV entries take priority — never overwritten)
- [ ] Phase 1: Applies market cap pre-filter using cached tiers (fail-open: uncached tickers pass through)
- [ ] Phase 1: Logs enriched coverage: `"Metadata enrichment: sector_map=%d, industry_group_map=%d"`
- [ ] Phase 3 (`_process_ticker_options()`): After `fetch_ticker_info()`, calls `map_yfinance_to_metadata()` → `upsert_ticker_metadata()`
- [ ] Phase 3: Enriches `ticker_score.sector` if still `None` after Phase 1 assignment
- [ ] Phase 3: Enriches `ticker_score.industry_group` if still `None` after Phase 1 assignment
- [ ] Existing scan tests still pass

## Technical Details

### Phase 1 Integration (~line 270 of pipeline.py)
- After `industry_group_map` is built from S&P 500 CSV:
  ```python
  all_metadata = await self._repo.get_all_ticker_metadata()
  for meta in all_metadata:
      if meta.ticker not in sector_map and meta.sector is not None:
          sector_map[meta.ticker] = meta.sector
      if meta.ticker not in industry_group_map and meta.industry_group is not None:
          industry_group_map[meta.ticker] = meta.industry_group
  ```
- Market cap pre-filter: Build `market_cap_map: dict[str, MarketCapTier]` from metadata, apply filter if `--market-cap` flag is set (fail-open: tickers not in map pass through)
- Insert point: after line 270, before custom tickers branch at line 272

### Phase 3 Integration (~line 810 of pipeline.py)
- After `ticker_score.company_name = ticker_info.company_name`:
  ```python
  metadata = map_yfinance_to_metadata(ticker_info)
  await self._repo.upsert_ticker_metadata(metadata)
  if ticker_score.sector is None and metadata.sector is not None:
      ticker_score.sector = metadata.sector
  if ticker_score.industry_group is None and metadata.industry_group is not None:
      ticker_score.industry_group = metadata.industry_group
  ```
- `ScanPipeline` already has `self._repo: Repository` — no new DI needed

### Edge Cases
- Empty metadata table → Phase 1 proceeds with S&P 500 data only
- Metadata has sector but not industry_group → only sector_map extended
- Phase 3 upsert fails (DB error) → logged, does not crash scan
- Ticker in both S&P 500 CSV and metadata → CSV value preserved
