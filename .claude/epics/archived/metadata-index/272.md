# Task: universe index CLI command

## Description
Add `options-arena universe index [--force] [--concurrency 5] [--max-age 30]` subcommand to `cli/commands.py`. Fetches yfinance `Ticker.info` for all stale/missing CBOE tickers, maps to `TickerMetadata`, and persists to SQLite. Includes Rich progress bar and final coverage report.

## Acceptance Criteria
- [ ] `universe index` subcommand added to `universe_app` in `cli/commands.py`
- [ ] `--force` flag re-indexes all tickers regardless of staleness
- [ ] `--concurrency` option (default 5) controls concurrent yfinance calls via `asyncio.Semaphore`
- [ ] `--max-age` option (default 30) controls staleness threshold in days
- [ ] Rich progress bar shows ticker count and completion percentage
- [ ] Individual ticker failures are caught, logged, and skipped (never crash batch)
- [ ] Final report shows: indexed count, failed count, sector coverage %, unmapped industry strings
- [ ] Uses sync Typer wrapper + `asyncio.run()` pattern
- [ ] Uses existing `RateLimiter` for yfinance throttling

## Technical Details
- Add `@universe_app.command()` after the existing `stats` command (~line 995)
- Sync wrapper: `def index(...) -> None: asyncio.run(_index_async(...))`
- Async implementation:
  1. Create services (`MarketDataService`, `UniverseService`, `Database`, `Repository`)
  2. Fetch CBOE ticker list via `universe.fetch_optionable_tickers()`
  3. If not `--force`, get stale tickers via `repo.get_stale_tickers(max_age)`
  4. Iterate with `asyncio.Semaphore(concurrency)` for controlled parallelism
  5. For each ticker: `fetch_ticker_info()` → `map_yfinance_to_metadata()` → `upsert_ticker_metadata()`
  6. Catch per-ticker exceptions, log, continue
  7. Final: `repo.get_metadata_coverage()` → Rich table with stats
- Use `rich.progress.Progress` for the progress bar (existing pattern in CLI)
- Close all services in `finally` block

### Edge Cases
- Empty CBOE universe → "No tickers to index" message
- All tickers fresh (not stale) without --force → "All tickers up to date"
- yfinance returns empty info dict → ticker logged as failed, skipped
- KeyboardInterrupt during indexing → graceful shutdown via signal handler
