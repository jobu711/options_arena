# Task: Agent Integration

## Description
Pass `SpreadAnalysis` data to debate agents via `DebateDeps` and `MarketContext`. Add flat spread fields to `MarketContext` for prompt rendering. Enrich Volatility and Risk agent prompts with actual spread data (replacing LLM guess). Set `TradeThesis.recommended_strategy` from the algorithmic engine's result, giving it precedence over the LLM's suggestion.

## Acceptance Criteria
- [ ] `DebateDeps.spread_analysis: SpreadAnalysis | None = None` field added
- [ ] Flat spread fields added to `MarketContext` model
- [ ] Volatility agent prompt enriched with spread type, net premium, breakevens
- [ ] Risk agent prompt enriched with max_loss, risk/reward ratio, PoP
- [ ] `TradeThesis.recommended_strategy` set from algorithmic result
- [ ] `_render_optional()` guards on all new fields (no prompt bloat when None)
- [ ] Algorithmic recommendation takes priority over LLM suggestion
- [ ] No regressions in existing agent tests
- [ ] `mypy --strict` clean

### MarketContext New Fields
In `models/analysis.py`:
```python
class MarketContext(BaseModel):
    # existing fields...

    # Spread strategy fields (flat for agent prompt parsing)
    spread_type: SpreadType | None = None
    spread_net_premium: Decimal | None = None
    spread_max_profit: Decimal | None = None
    spread_max_loss: Decimal | None = None
    spread_pop: float | None = None
    spread_risk_reward: float | None = None
```
Add `field_serializer` for new Decimal fields. Add to existing `serialize_decimal`.

### Prompt Enrichment
**Volatility agent** (`agents/prompts/volatility.py`):
```
## Algorithmic Strategy Recommendation
Strategy: {spread_type}
Net Premium: ${spread_net_premium}
Breakevens: {breakevens}
PoP: {spread_pop:.1%}
```

**Risk agent** (`agents/prompts/risk.py`):
```
## Spread Risk Profile
Strategy: {spread_type}
Max Loss: ${spread_max_loss}
Max Profit: ${spread_max_profit}
Risk/Reward: {spread_risk_reward:.2f}
PoP: {spread_pop:.1%}
```

Both sections guarded by `if spread_type is not None:` and `_render_optional()` pattern.

### Algorithmic Precedence
In orchestrator, after all agents run:
```python
# Set recommended_strategy from algorithmic engine (priority over LLM)
if deps.spread_analysis is not None:
    thesis = thesis.model_copy(
        update={"recommended_strategy": deps.spread_analysis.spread.spread_type}
    )
```

### Domain Context Partitioning
- Spread data goes to Volatility and Risk agents only (domain-relevant)
- Bull, Bear, Trend, Flow, Fundamental, Contrarian agents do NOT see spread fields
- Follows existing partitioning pattern (render_trend_context, render_volatility_context, etc.)
