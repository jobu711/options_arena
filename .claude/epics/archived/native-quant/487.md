## Technical Details

### Formulas
**Parkinson (1980)**: `sigma^2 = (1/(4n*ln2)) * Sum(ln(H/L))^2` -> annualize with `sqrt(252)`
**Rogers-Satchell (1991)**: `sigma^2 = (1/n) * Sum[ln(H/C)*ln(H/O) + ln(L/C)*ln(L/O)]` -> annualize with `sqrt(252)`
**Yang-Zhang (2000)**: `sigma^2_yz = sigma^2_overnight + k*sigma^2_close + (1-k)*sigma^2_rs` where:
- `sigma^2_overnight = (1/(n-1)) * Sum(ln(O_i/C_{i-1}) - mean)^2`
- `sigma^2_close = (1/(n-1)) * Sum(ln(C_i/O_i) - mean)^2`
- `sigma^2_rs` = Rogers-Satchell variance (un-annualized)
- `k = 0.34 / (1.34 + (n+1)/(n-1))`

### Key Rules
- Use `np.log()` for vectorized log on Series, NOT `math.log()`
- Division-by-zero guards on all ratios (`replace(0.0, np.nan)`)
- `ddof=1` for sample variance (Yang-Zhang overnight/close components)
- Input validation: all multi-Series functions must check aligned lengths
- All functions use `pd.Series` inputs only (indicators module convention)
