# Task: ChainProvider Protocol + YFinanceChainProvider

## Description

Define a `ChainProvider` runtime `Protocol` in `services/options_data.py` and extract existing yfinance chain-fetching logic into a `YFinanceChainProvider` class implementing that protocol. The `OptionsDataService` public API (`fetch_chain`, `fetch_expirations`, `fetch_chain_all_expirations`) remains unchanged — this is a pure internal refactor.

## Acceptance Criteria

- [ ] `ChainProvider` protocol defined with `fetch_chain(ticker, expiration) -> list[OptionContract]` and `fetch_expirations(ticker) -> list[date]`
- [ ] `YFinanceChainProvider` class extracts all yfinance-specific logic from `OptionsDataService`
- [ ] `YFinanceChainProvider` implements `ChainProvider` protocol (runtime checkable)
- [ ] `OptionsDataService` delegates to `YFinanceChainProvider` internally
- [ ] All existing `test_options_data.py` tests pass without modification
- [ ] No behavioral change — identical output for all inputs

## Technical Details

### Files Modified
- `src/options_arena/services/options_data.py` — add `ChainProvider` protocol, extract `YFinanceChainProvider`

### Implementation Approach

1. Define protocol at module level:
   ```python
   from typing import Protocol, runtime_checkable

   @runtime_checkable
   class ChainProvider(Protocol):
       async def fetch_expirations(self, ticker: str) -> list[date]: ...
       async def fetch_chain(self, ticker: str, expiration: date) -> list[OptionContract]: ...
   ```

2. Extract `YFinanceChainProvider` class:
   - Moves `_yf_call()`, cache logic, yfinance DataFrame parsing into the class
   - Constructor: `__init__(self, config: ServiceConfig, pricing_config: PricingConfig, cache: ServiceCache, limiter: RateLimiter)`
   - Reuses existing module-level helpers: `_passes_liquidity_filter`, `_row_to_contract`, `_contracts_to_cache_bytes`, `_cache_bytes_to_contracts`

3. `OptionsDataService` stores one `YFinanceChainProvider` instance internally and delegates:
   ```python
   class OptionsDataService:
       def __init__(self, config, pricing_config, cache, limiter):
           self._yf_provider = YFinanceChainProvider(config, pricing_config, cache, limiter)

       async def fetch_chain(self, ticker, expiration):
           return await self._yf_provider.fetch_chain(ticker, expiration)
   ```

### Key Constraints
- Module-level helpers (`_passes_liquidity_filter`, `_row_to_contract`, etc.) stay as module functions — don't move into the class (they're pure functions, no state)
- `ExpirationChain` model stays in the module
- `close()` delegates to provider if provider has resources to close
