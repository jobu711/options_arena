# Task: Frontend enrichment types + display

## Description

Extend the TypeScript `DebateResult` interface with enrichment fields and add 3 conditional enrichment sections to `DebateResultPage.vue`: Fundamental Profile, Unusual Flow, and News Sentiment. Add `enrichment_ratio` to the metadata strip.

## Acceptance Criteria

- [ ] `DebateResult` interface in `debate.ts` has 12 optional enrichment fields matching API schema
- [ ] `DebateResultPage.vue` shows Fundamental Profile section when `pe_ratio` is present
- [ ] `DebateResultPage.vue` shows Unusual Flow section when `net_call_premium` is present
- [ ] `DebateResultPage.vue` shows News Sentiment section when `news_sentiment_score` is present
- [ ] All 3 sections hidden when enrichment data is absent (backward compatible)
- [ ] `enrichment_ratio` displayed in metadata strip (conditional, like `citation_density`)
- [ ] Styling matches existing dark theme (PrimeVue CSS vars)

## Technical Details

- `web/src/pages/DebateResultPage.vue` — Add enrichment sections + metadata item
