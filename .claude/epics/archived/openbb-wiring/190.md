### Test file locations follow project convention:

### Key test patterns (from tests/CLAUDE.md):
- **Never hit real APIs**: Mock OpenBBService constructor + fetch methods
- **Never use `==` for floats**: Use `pytest.approx()` for enrichment_ratio
- **Typed Pydantic models**: Use `FundamentalSnapshot(...)`, `UnusualFlowSnapshot(...)`, `NewsSentimentSnapshot(...)` as fixtures
- **AsyncMock**: All fetch methods are async — use `AsyncMock(return_value=...)`

### Mocking strategy:
```python
# Mock OpenBBService to avoid SDK import
mock_openbb = AsyncMock(spec=OpenBBService)
mock_openbb.fetch_fundamentals = AsyncMock(return_value=FundamentalSnapshot(...))
mock_openbb.fetch_unusual_flow = AsyncMock(return_value=UnusualFlowSnapshot(...))
mock_openbb.fetch_news_sentiment = AsyncMock(return_value=NewsSentimentSnapshot(...))
mock_openbb.close = AsyncMock()
```
