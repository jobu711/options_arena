# Task: BAW unit tests

## Description
Write comprehensive unit tests for `src/options_arena/pricing/american.py` in
`tests/unit/pricing/test_american.py`. ~40 tests covering Hull textbook reference values,
the FR-P4 call identity (BAW call = BSM call when q=0), FR-P5 put dominance
(BAW put >= BSM put always), finite-difference Greeks correctness, and IV round-trip.

These tests verify the core project differentiator: correct American options pricing.

## Acceptance Criteria
- [ ] `tests/unit/pricing/test_american.py` created with ~40 tests
- [ ] **FR-P4 identity**: `american_call ≈ bsm_call` when `q=0` across parameter grid (tight tolerance)
- [ ] **FR-P5 dominance**: `american_put >= bsm_put` for all parameter combinations (no exceptions)
- [ ] **Hull reference values**: at least 3 known BAW prices verified (parametrized)
- [ ] **Early exercise premium**: `american_put - bsm_put > 0` for deep ITM puts with dividends
- [ ] **Greeks sign correctness**: delta, gamma, vega, rho ranges match BSM conventions
- [ ] **Greeks `pricing_model`**: all Greeks instances have `pricing_model=PricingModel.BAW`
- [ ] **IV round-trip**: `american_price(american_iv(price)) ≈ price` within tolerance
- [ ] **IV convergence**: `american_iv` converges within `maxiter` for reasonable inputs
- [ ] **Boundary T→0**: price converges to intrinsic value
- [ ] **Boundary q=0**: call has zero early exercise premium
- [ ] Parametrized across `OptionType.CALL` / `OptionType.PUT`
- [ ] All tests use `pytest.approx(rel=1e-6)` or `pytest.approx(abs=0.01)` as appropriate
- [ ] No mocking — pure math functions
- [ ] `uv run pytest tests/unit/pricing/test_american.py -v` passes

## Technical Details

### Test categories (~40 tests total)

1. **FR-P4 — Call identity when q=0** (~6 tests)
   ```python
   @pytest.mark.parametrize("S,K,T,r,sigma", [
       (100, 100, 1.0, 0.05, 0.20),
       (100, 110, 0.5, 0.08, 0.30),
       (50, 45, 0.25, 0.03, 0.25),
       (200, 200, 2.0, 0.04, 0.15),
   ])
   def test_call_equals_bsm_when_no_dividend(S, K, T, r, sigma):
       q = 0.0
       bsm = bsm_price(S, K, T, r, q, sigma, OptionType.CALL)
       baw = american_price(S, K, T, r, q, sigma, OptionType.CALL)
       assert baw == pytest.approx(bsm, rel=1e-6)
   ```

2. **FR-P5 — Put dominance** (~6 tests)
   ```python
   def test_american_put_geq_bsm_put(S, K, T, r, q, sigma):
       bsm = bsm_price(S, K, T, r, q, sigma, OptionType.PUT)
       baw = american_price(S, K, T, r, q, sigma, OptionType.PUT)
       assert baw >= bsm - 1e-10  # small tolerance for float arithmetic
   ```

3. **Hull reference values** (~4 tests)
   - Parametrized with known BAW prices from Hull textbook
   - Absolute tolerance (prices verified to 2 decimal places)

4. **Early exercise premium** (~4 tests)
   - Deep ITM puts with q > 0 should show meaningful premium
   - Premium increases with higher r (more incentive to exercise early)

5. **Greeks correctness** (~10 tests)
   - Sign correctness for all 5 Greeks (parametrized by call/put)
   - `pricing_model == PricingModel.BAW` on every OptionGreeks instance
   - Gamma > 0 for both call and put
   - Compare Greek magnitudes to BSM Greeks (should be similar for OTM options)

6. **IV solver** (~6 tests)
   - Round-trip across parameter grid
   - Convergence within maxiter
   - ValueError for impossible market prices
   - Respects PricingConfig tolerance

7. **Boundary conditions** (~4 tests)
   - T→0 returns intrinsic
   - q=0 call identity (cross-check with FR-P4)
   - Deep ITM / deep OTM behavior
