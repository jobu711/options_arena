## Technical Details

### `dispatch.py` structure
```python
"""Unified pricing dispatch by ExerciseStyle."""
import logging
from options_arena.models.enums import ExerciseStyle, OptionType
from options_arena.models.options import OptionGreeks
from options_arena.models.config import PricingConfig
from options_arena.pricing.bsm import bsm_price, bsm_greeks, bsm_iv
from options_arena.pricing.american import american_price, american_greeks, american_iv

def option_price(
    exercise_style: ExerciseStyle,
    S: float, K: float, T: float, r: float, q: float,
    sigma: float, option_type: OptionType,
) -> float:
    match exercise_style:
        case ExerciseStyle.AMERICAN:
            return american_price(S, K, T, r, q, sigma, option_type)
        case ExerciseStyle.EUROPEAN:
            return bsm_price(S, K, T, r, q, sigma, option_type)

def option_greeks(
    exercise_style: ExerciseStyle,
    S: float, K: float, T: float, r: float, q: float,
    sigma: float, option_type: OptionType,
) -> OptionGreeks:
    match exercise_style:
        case ExerciseStyle.AMERICAN:
            return american_greeks(S, K, T, r, q, sigma, option_type)
        case ExerciseStyle.EUROPEAN:
            return bsm_greeks(S, K, T, r, q, sigma, option_type)

def option_iv(
    exercise_style: ExerciseStyle,
    market_price: float,
    S: float, K: float, T: float, r: float, q: float,
    option_type: OptionType,
    config: PricingConfig | None = None,
) -> float:
    if config is None:
        config = PricingConfig()
    match exercise_style:
        case ExerciseStyle.AMERICAN:
            return american_iv(market_price, S, K, T, r, q, option_type, config)
        case ExerciseStyle.EUROPEAN:
            return bsm_iv(market_price, S, K, T, r, q, option_type)
```

### `__init__.py` update

```python
"""Options Arena — Options pricing (BSM, BAW) and Greeks computation."""

from options_arena.pricing.dispatch import option_greeks, option_iv, option_price

__all__ = ["option_greeks", "option_iv", "option_price"]
```

### Design decisions
- `config` param defaults to `PricingConfig()` if not provided (production defaults)
- BSM IV doesn't need full config (tolerance/maxiter are hardcoded in Newton-Raphson) — pass through initial_guess only
- Match is exhaustive over `ExerciseStyle` — mypy will catch missing cases
- No business logic in dispatch — pure routing only
