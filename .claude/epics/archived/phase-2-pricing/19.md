## Technical Details

### Test categories (~20 tests total)

1. **Routing correctness** (~6 tests)
   - `option_price(AMERICAN, ...)` matches `american_price(...)` exactly
   - `option_price(EUROPEAN, ...)` matches `bsm_price(...)` exactly
   - Same for `option_greeks` and `option_iv`

2. **Greeks metadata** (~4 tests)
   - AMERICAN routing → `pricing_model == PricingModel.BAW`
   - EUROPEAN routing → `pricing_model == PricingModel.BSM`
   - Both call and put

3. **Config handling** (~4 tests)
   - `option_iv` with explicit `PricingConfig`
   - `option_iv` with `config=None` (default)
   - Custom tolerance respected

4. **Package re-export** (~3 tests)
   - `from options_arena.pricing import option_price` works
   - `from options_arena.pricing import option_greeks` works
   - `from options_arena.pricing import option_iv` works

5. **Cross-model consistency** (~3 tests)
   - For European style, dispatch result == direct BSM result
   - For American style with q=0 call, dispatch result ≈ BSM result (FR-P4 via dispatch)
   - For American put, dispatch result >= BSM result (FR-P5 via dispatch)

### Verification gate

```bash
uv run ruff check . --fix && uv run ruff format .
uv run pytest tests/ -v
uv run mypy src/ --strict
```

All three must pass. If any existing tests break (regression), fix before marking complete.
