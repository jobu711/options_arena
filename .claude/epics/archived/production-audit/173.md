## Technical Details

### WebSocket Close Defense (AUDIT-026)
```python
try:
    await websocket.close()
except Exception:
    pass  # Connection already closed
```
Apply to all `websocket.close()` calls in `api/ws.py` (3 endpoints).

### Exception Hierarchy Tests (AUDIT-029)
```python
def test_ticker_not_found_is_data_fetch_error() -> None:
```
