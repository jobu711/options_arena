### New classify function
```python
def classify_macd_signal(macd_value: float | None) -> MacdSignal:
    """Classify raw MACD histogram value into a signal.

    Args:
        macd_value: Raw (un-normalized) MACD histogram value.

    Returns:
        MacdSignal based on histogram sign.
    """
    if macd_value is None or not math.isfinite(macd_value):
        return MacdSignal.NEUTRAL
    if macd_value > 0:
        return MacdSignal.BULLISH_CROSSOVER
    if macd_value < 0:
        return MacdSignal.BEARISH_CROSSOVER
    return MacdSignal.NEUTRAL
```

### Update build_market_context()
- Currently calls `macd_signal = _derive_macd_signal(ticker_score.direction)`
- Change to: `macd_signal = classify_macd_signal(ticker_score.signals.macd)`
- Note: `ticker_score.signals.macd` holds the **normalized** value (0-100). For classification, we need the raw value. Check if raw signals are available via `TickerScore` or passed separately. If only normalized is available, adjust: normalized > 50 → positive raw, < 50 → negative raw. But ideally source from raw signals dict.
- If raw signals not easily available, the normalized value works: > 50 maps to positive histogram (bullish), < 50 maps to negative (bearish), ~50 is neutral.
