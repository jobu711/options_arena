# Task: Create PresetCard.vue component

## Description
Create a new `web/src/components/scan/PresetCard.vue` component — a clickable card that represents one scan preset. Displays icon, preset name, description, and estimated ticker count. Supports selected, hover, and disabled visual states.

## Acceptance Criteria
- [ ] Component renders icon, label, description, and count
- [ ] Click emits `select` event with preset string
- [ ] Selected state: `border-color: var(--accent-blue)`, `background: rgba(59, 130, 246, 0.08)`
- [ ] Hover state: `border-color: var(--p-surface-500)`
- [ ] Disabled state: `opacity: 0.5`, `pointer-events: none`
- [ ] `data-testid="preset-card-{preset}"` on root element
- [ ] Count displayed via PrimeVue Badge (severity="secondary")
- [ ] Transition: `0.15s` on border-color and background
- [ ] All `<style scoped>`, uses existing CSS custom properties only
