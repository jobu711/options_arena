# Task: Migrate IntelligenceService and FinancialDatasetsService to ServiceBase

## Description
Migrate `IntelligenceService` and `FinancialDatasetsService` to inherit from `ServiceBase`.
Both are moderate complexity: IntelligenceService uses `_yf_call` + never-raises pattern;
FinancialDatasetsService uses httpx + cache + limiter. Constructor signatures must remain
backward-compatible. All existing tests must pass without modification.

## Acceptance Criteria
- [ ] `IntelligenceService` inherits `ServiceBase[IntelligenceConfig]`
- [ ] `IntelligenceService.__init__` calls `super().__init__(config, cache, limiter)`
- [ ] IntelligenceService raw `asyncio.to_thread()` calls migrated to `self._yf_call()`
- [ ] IntelligenceService cache blocks replaced with `self._cached_fetch()` (5 methods)
- [ ] IntelligenceService retry calls use `self._retried_fetch()`
- [ ] IntelligenceService never-raise contract preserved
- [ ] `FinancialDatasetsService` inherits `ServiceBase[FinancialDatasetsConfig]`
- [ ] `FinancialDatasetsService.__init__` calls `super().__init__(config, cache, limiter)`
- [ ] FinancialDatasetsService cache blocks replaced with `self._cached_fetch()` (3 methods)
- [ ] FinancialDatasetsService retry calls use `self._retried_fetch()`
- [ ] `FinancialDatasetsService.close()` still calls `self._client.aclose()`
- [ ] Constructor signatures unchanged — backward compatible
- [ ] All existing tests pass unchanged
- [ ] No consumer code changes
- [ ] Lint, format, and type check pass

## Technical Details

### IntelligenceService Migration
- Config type: `IntelligenceConfig` (has `request_timeout: float`, NOT `yfinance_timeout`)
- Has raw `asyncio.to_thread()` calls → migrate to `self._yf_call(fn, *args, timeout=self._config.request_timeout)`
- 5 cached methods use direct model serde → convert to `_cached_fetch()`
- Uses `_retried_fetch` pattern → delegate to `self._retried_fetch()`
- Never-raise contract: all methods return `None` on error
- No httpx client → default `close()` no-op is correct

### FinancialDatasetsService Migration
- Config type: `FinancialDatasetsConfig` (has `request_timeout: float`)
- 3 cached methods → convert to `_cached_fetch()`
- Uses retry pattern → delegate to `self._retried_fetch()`
- Has httpx client → override `close()` to call `self._client.aclose()`
- `self._client` stays in subclass `__init__`

- `src/options_arena/services/intelligence_service.py` (modify)
