### Implementation Notes
- Route handler is a thin join â€” no business logic:
  1. `constituents = await universe.fetch_sp500_constituents()`
  2. `tickers = [c.ticker for c in constituents]`
  3. `quotes = await market_data.fetch_batch_daily_changes(tickers)`
  4. `metadata = await repo.get_all_ticker_metadata()`
  5. Build lookup dicts, join into `list[HeatmapTicker]`
- `MARKET_CAP_WEIGHTS: dict[str, float]` constant in route file maps `MarketCapTier` values to weights
- Use `SP500Constituent.sector` as fallback when `TickerMetadata.sector` is None
- Router prefix: `/api/market`, tags: `["market"]`
- Follow `routes/universe.py` pattern for imports, limiter usage, response model annotation
