# Task: Export — 4 v2 section renderers + protocol-aware export

## Description
Add 4 markdown section renderer functions for v2 agent outputs (flow, fundamental, risk_v2, contrarian). Modify `export_debate_markdown()` to include v2 sections when `debate_protocol == "v2"`, rename "Bull Case" to "Trend Analysis" for v2, and omit synthetic "Bear Case" for v2.

- [ ] `_render_flow_section(flow: FlowThesis)` returns markdown with GEX, smart money, OI, volume, key factors
- [ ] `_render_fundamental_section(fund: FundamentalThesis)` returns markdown with catalyst, earnings, IV crush, key factors
- [ ] `_render_risk_v2_section(risk: RiskAssessment)` returns markdown with risk level, PoP, max loss, risks, mitigants
- [ ] `_render_contrarian_section(contra: ContrarianThesis)` returns markdown with dissent, challenge, overlooked risks, alt scenario

  - V2: Trend Analysis → Flow Analysis → Fundamental Analysis → Volatility → Risk Assessment → Contrarian Challenge → Verdict
  - V1: Bull Case → Bear Case → (Rebuttal) → (Volatility) → Verdict (unchanged)
- [ ] V2 uses "Trend Analysis" heading instead of "Bull Case"
- [ ] V2 omits "Bear Case" section
- [ ] V1 export is identical to before (zero regression)
- [ ] `mypy --strict` passes
