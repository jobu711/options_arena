### Scheduler Method (`services/outcome_collector.py`)
```python
async def run_scheduler(self) -> None:
    """Run daily outcome collection on schedule. Never raises."""
    while True:
        try:
            seconds = self._seconds_until_next_run()
            logger.info("Next outcome collection in %.0f seconds", seconds)
            await asyncio.sleep(seconds)
            result = await self.collect_outcomes()
            logger.info("Auto-collected: %d contracts", result.total_collected)
        except asyncio.CancelledError:
            logger.info("Outcome scheduler cancelled")
            break
        except Exception:
            logger.exception("Scheduler error — will retry next cycle")
```

### Lifespan Wiring (`api/app.py`)
```python
# In lifespan(), after creating collector:
scheduler_task = None
if settings.analytics.auto_collect_enabled:
    scheduler_task = asyncio.create_task(collector.run_scheduler())

yield

if scheduler_task is not None:
    scheduler_task.cancel()
    with suppress(asyncio.CancelledError):
        await scheduler_task
```

### Edge Cases
- Target hour already passed today: schedule for tomorrow
- Exactly at target hour: schedule for tomorrow (avoid double-run)
- Collection error: log and continue (never-raises contract)
- Server restart mid-collection: partial outcomes are safe (per-contract saves)
- CancelledError during sleep: clean exit

### Mocking Strategy
- Mock `asyncio.sleep()` to avoid real waiting
- Mock `datetime.now(UTC)` to control time calculations
- Mock `collect_outcomes()` to verify it's called
- Use `asyncio.CancelledError` to test shutdown path
