# Task: Implement _batch_async() orchestration loop

## Description
Implement the `_batch_async()` function in `src/options_arena/cli/commands.py` that:
1. Loads the latest scan from the database
2. Retrieves the top N ticker scores (sorted by composite_score descending)
3. Creates services ONCE (shared across all tickers)
4. Iterates over each ticker, calling `_debate_single()` with error isolation
5. Collects results (success or failure per ticker)
6. Renders the batch summary table at the end
7. Properly closes all services in a `finally` block

## Acceptance Criteria
- [ ] `_batch_async()` function implemented with signature: `async def _batch_async(batch_limit: int, fallback_only: bool) -> None`
- [ ] Loads latest scan from Repository; errors clearly if no scan exists
- [ ] Sorts scores by `composite_score` descending, takes top `batch_limit`
- [ ] Services (MarketDataService, OptionsDataService, FredService, Database, ServiceCache) created once
- [ ] Each ticker debate wrapped in try/except — one failure does NOT stop others
- [ ] Per-ticker status printed to stderr: `[cyan]Debating AAPL (1/5)...[/cyan]`
- [ ] Failed tickers logged with error message, included in summary
- [ ] After all debates: `render_batch_summary_table()` called (from rendering.py)
- [ ] Token usage + total duration shown at the end
- [ ] Regulatory disclaimer printed at the end
- [ ] All services closed in `finally` block
- [ ] `ruff check`, `ruff format`, `mypy --strict` all pass

## Technical Details

```python
async def _batch_async(batch_limit: int, fallback_only: bool) -> None:
    settings = AppSettings()
    _DATA_DIR.mkdir(parents=True, exist_ok=True)
    cache = ServiceCache(settings.service)
    limiter = RateLimiter(...)
    db = Database(...)

    market_data: MarketDataService | None = None
    options_data: OptionsDataService | None = None
    fred: FredService | None = None

    try:
        await db.connect()
        repo = Repository(db)

        # Load latest scan scores
        latest_scan = await repo.get_latest_scan()
        if latest_scan is None:
            err_console.print("[red]No scan data. Run: options-arena scan --preset sp500[/red]")
            raise typer.Exit(code=1)

        scores = await repo.get_scores_for_scan(latest_scan.id)
        top_scores = sorted(scores, key=lambda s: s.composite_score, reverse=True)[:batch_limit]

        if not top_scores:
            err_console.print("[yellow]No scored tickers in latest scan.[/yellow]")
            return

        # Create services once
        market_data = MarketDataService(...)
        options_data = OptionsDataService(...)
        fred = FredService(...)

        err_console.print(f"[cyan]Batch debate: {len(top_scores)} tickers[/cyan]\n")

        results: list[tuple[str, DebateResult | None, str | None]] = []
        start_time = time.monotonic()

        for i, ticker_score in enumerate(top_scores, 1):
            ticker = ticker_score.ticker
            err_console.print(f"[cyan]Debating {ticker} ({i}/{len(top_scores)})...[/cyan]")
            try:
                result = await _debate_single(
                    ticker_score, settings, market_data, options_data, fred, repo,
                    fallback_only=fallback_only,
                )
                results.append((ticker, result, None))
            except Exception as exc:
                logger.exception("Batch debate failed for %s", ticker)
                results.append((ticker, None, str(exc)))

        # Render summary
        elapsed = time.monotonic() - start_time
        table = render_batch_summary_table(results)
        console.print(table)
        console.print(f"\n[dim]Total: {len(results)} tickers in {elapsed:.1f}s[/dim]")
        console.print(f"\n{DISCLAIMER}")

    except KeyboardInterrupt:
        err_console.print("\n[yellow]Batch debate cancelled.[/yellow]")
        raise typer.Exit(code=130)
    except typer.Exit:
        raise
    except Exception as exc:
        logger.exception("Batch debate failed")
        err_console.print("[red]Batch failed. Check logs/options_arena.log[/red]")
        raise typer.Exit(code=1) from exc
    finally:
        if market_data is not None:
            await market_data.close()
        if options_data is not None:
            await options_data.close()
        if fred is not None:
            await fred.close()
        await cache.close()
        await db.close()
```

Key design decisions:
- Services created once, shared across all tickers (as specified in epic)
- Sequential execution (not parallel) — LLM can only handle one debate at a time
- Error isolation via per-ticker try/except
- Results collected as `list[tuple[str, DebateResult | None, str | None]]`
