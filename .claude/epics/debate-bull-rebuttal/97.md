# Task: Update risk agent to inject bull rebuttal context

## Description
Update the risk agent's dynamic prompt to include the bull's rebuttal when
`ctx.deps.bull_rebuttal` is set. This gives the risk agent full context of the debate:
bull's initial case, bear's counter, and bull's defense.

## Acceptance Criteria
- [ ] When `ctx.deps.bull_rebuttal is not None`: prompt includes `<<<BULL_REBUTTAL>>>...<<<END_BULL_REBUTTAL>>>` block
- [ ] Rebuttal block includes: direction, confidence, argument, key_points (same structure as existing bull/bear blocks)
- [ ] When `ctx.deps.bull_rebuttal is None`: prompt unchanged (no rebuttal block)
- [ ] Block inserted after the bear case block and before the vol case block (if present)

## Technical Details
- File: `src/options_arena/agents/risk.py`
- Add a new conditional block in `risk_dynamic_prompt()` after the bear case injection:
  ```python
  if ctx.deps.bull_rebuttal is not None:
      parts.append(
          f"\n\n<<<BULL_REBUTTAL>>>\n"
          f"Direction: {ctx.deps.bull_rebuttal.direction.value}\n"
          f"Confidence: {ctx.deps.bull_rebuttal.confidence}\n"
          f"Argument: {ctx.deps.bull_rebuttal.argument}\n"
          f"Key Points: {', '.join(ctx.deps.bull_rebuttal.key_points)}\n"
          f"<<<END_BULL_REBUTTAL>>>"
      )
  ```
- Follows the exact same pattern as the existing bull_case and bear_case blocks
- No changes to risk agent init, output_validator, or system prompt constant
