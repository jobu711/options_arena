# Task: Add rebuttal panel to CLI rendering

## Description
Add a green italic Rich panel for the bull's rebuttal in `render_debate_panels()`.
The panel appears after the bear panel and before the volatility panel (if present)
or the verdict panel. Only rendered when `result.bull_rebuttal` is not None.

## Acceptance Criteria
- [ ] When `result.bull_rebuttal is not None`: green-bordered panel rendered with title "BULL REBUTTAL"
- [ ] Panel uses `_build_agent_panel_text()` (same helper as bull/bear panels)
- [ ] Panel border_style is "green" (same as bull, since it's the bull defending)
- [ ] Panel title is "BULL REBUTTAL" and title_align is "left"
- [ ] Panel appears after bear panel, before volatility panel (if present)
- [ ] When `result.bull_rebuttal is None`: no panel rendered (no extra whitespace)
- [ ] Text uses `Text()` constructor (no markup) to prevent bracket interpretation

## Technical Details
- File: `src/options_arena/cli/rendering.py`
- Insert rebuttal rendering block between the bear panel and volatility panel sections
- Reuse `_build_agent_panel_text()` which already handles direction, confidence, argument,
  key_points, and risks formatting
- Pattern:
  ```python
  # --- Bull rebuttal panel (optional) ---
  if result.bull_rebuttal is not None:
      rebuttal = result.bull_rebuttal
      rebuttal_body = _build_agent_panel_text(
          direction=rebuttal.direction.value.upper(),
          confidence=rebuttal.confidence,
          argument=rebuttal.argument,
          key_points=rebuttal.key_points,
          risks=rebuttal.risks_cited,
      )
      console.print(
          Panel(rebuttal_body, border_style="green", title="BULL REBUTTAL", title_align="left")
      )
      console.print()
  ```
