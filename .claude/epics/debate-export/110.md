# Task: Implement file export with PDF support

## Description
Add `export_debate_to_file()` and `_render_pdf()` to `src/options_arena/reporting/debate_export.py`.
The file export function writes markdown or PDF to disk. PDF rendering uses `weasyprint` as an
optional dependency — if not installed, raise `ImportError` with install instructions.

Also add the `weasyprint` optional dependency group to `pyproject.toml`.

## Acceptance Criteria
- [ ] `export_debate_to_file(result: DebateResult, path: Path, fmt: str = "md") -> Path` implemented
- [ ] When `fmt="md"`, writes markdown string to file with UTF-8 encoding, returns path
- [ ] When `fmt="pdf"`, delegates to `_render_pdf()`, returns path
- [ ] Raises `ValueError` for unsupported format strings
- [ ] `_render_pdf(md_content: str, path: Path) -> Path` wraps markdown in minimal HTML, renders via weasyprint
- [ ] `ImportError` raised with clear install message when weasyprint not available
- [ ] `pyproject.toml` updated with `[project.optional-dependencies]` pdf group
- [ ] Re-exports updated in `__init__.py`
- [ ] Passes `ruff check`, `ruff format`, and `mypy --strict`

## Technical Details
- **File**: `src/options_arena/reporting/debate_export.py` (extend existing)
- **PDF rendering**: Wrap markdown in `<html><body><pre>{md_content}</pre></body></html>`
- **weasyprint import**: Lazy import inside `_render_pdf()` with `# type: ignore[import-untyped]`
- **pyproject.toml**: Add under `[project.optional-dependencies]`:
  ```toml
  pdf = ["weasyprint>=63.0"]
  ```
- **Error message**: `"PDF export requires weasyprint. Install: uv add 'options-arena[pdf]'"`
- `_render_pdf` is private (prefixed `_`), not re-exported
