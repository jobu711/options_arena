### Model schemas (from PRD Section 4)
- **AnalystSnapshot**: ticker, target_low/high/mean/median/current (float|None), strong_buy/buy/hold/sell/strong_sell (int), consensus_score (float|None computed), target_upside_pct (float|None), fetched_at (datetime UTC)
- **UpgradeDowngrade**: firm (str), action (str — mapped via ACTION_MAP), to_grade (str), from_grade (str|None), date (date), price_target/prior_price_target (float|None)
- **AnalystActivitySnapshot**: ticker, recent_changes (list[UpgradeDowngrade] up to 10), upgrades_30d/downgrades_30d (int), net_sentiment_30d (int), fetched_at
- **InsiderTransaction**: insider_name, position, transaction_type (parsed from Text), shares (int), value (float|None), ownership_type, transaction_date (date|None)
- **InsiderSnapshot**: ticker, transactions (list[InsiderTransaction] up to 20), net_insider_buys_90d (int), net_insider_value_90d (float|None), insider_buy_ratio (float|None [0,1]), fetched_at
- **InstitutionalSnapshot**: ticker, institutional_pct/institutional_float_pct/insider_pct (float|None [0,1]), institutions_count (int|None), top_holders (list[str] top 5), top_holder_pcts (list[float] top 5), fetched_at
- **IntelligencePackage**: ticker, analyst (AnalystSnapshot|None), analyst_activity (AnalystActivitySnapshot|None), insider (InsiderSnapshot|None), institutional (InstitutionalSnapshot|None), news_headlines (list[str]|None up to 5), fetched_at. Method: `intelligence_completeness() -> float`

### Key patterns
- Follow `models/CLAUDE.md`: frozen=True, NaN/Inf validators, UTC validators, no raw dicts
- Follow existing `FundamentalSnapshot` / `UnusualFlowSnapshot` patterns from OpenBB models
- Non-negative int validators on counts (strong_buy, buy, hold, sell, strong_sell, shares)
- `from_grade`: empty string → `None` conversion

### Edge Cases
- NaN/Inf on every float field → ValueError
- Non-UTC datetime on fetched_at → ValueError
- Zero total analysts → consensus_score is None
- Empty string from_grade → None
- Price target of 0.0 → None
- Transaction Text with no recognizable type → "Other"
- Buy ratio > 1.0 → ValueError
