## Technical Details
- Use `benchmark` fixture from pytest-benchmark
- Group benchmarks by module: `@pytest.mark.benchmark(group="pricing")`
- Representative inputs per group:
  - Pricing: S=100, K=100, T=30/365, r=0.05, q=0, σ=0.25 (ATM, 30 DTE)
  - Indicators: 250-row DataFrame with realistic OHLCV data
  - Scoring: 50-ticker universe with pre-computed indicator values
  - Orchestration: 6-agent confidence/direction arrays
- `--benchmark-autosave` stores results in `.benchmarks/`
- `--benchmark-compare` compares against previous run
- Baseline JSON stored in `tests/audit/reference_data/benchmarks/` for version control
