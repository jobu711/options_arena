# Task: Agent Discovery Skill — /math-audit Skill + Quant-Analyst Prompt Enhancement

## Description
Create the `/math-audit` CLI skill that invokes the quant-analyst agent to review mathematical functions for correctness issues. Enhance the quant-analyst agent prompt to support structured formula review with severity-classified findings. This is a non-blocking discovery tool — findings are advisory and require human review.

## Acceptance Criteria
- [ ] `.claude/commands/math-audit.md` — skill definition invoking quant-analyst agent
- [ ] Skill prompt instructs agent to: read source code, verify against cited papers, check edge cases, produce structured findings
- [ ] Output is a list of `AuditFinding` models with severity, function name, description, and proposed test
- [ ] `--discover` flag in `cli/audit.py` triggers the `/math-audit` flow
- [ ] Agent findings are printed to CLI with Rich formatting (severity-colored)
- [ ] Non-blocking: agent errors produce a warning, never crash the CLI
- [ ] Findings include proposed test code that humans can review and add to `tests/audit/`

## Technical Details
- Skill definition follows existing pattern in `.claude/commands/` directory
- The skill reads mathematical source files and asks quant-analyst to verify formulas
- Uses structured prompt: for each function, verify formula matches cited paper, check sign errors, check missing terms, check boundary conditions
- Output structured as `AuditFinding` models with `layer=AuditLayer.DISCOVERY`
- Severity classification: CRITICAL (wrong formula), WARNING (edge case not handled), INFO (undocumented approximation)
- Agent errors caught and logged — never blocks CLI
