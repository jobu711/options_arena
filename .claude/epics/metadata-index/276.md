# Task: map_yfinance_to_metadata() helper

## Description
Create `map_yfinance_to_metadata(ticker_info: TickerInfo) -> TickerMetadata` in `services/universe.py`. This standalone function resolves raw yfinance sector/industry strings to typed GICS enums using existing alias dicts, and logs unmapped strings at WARNING.

## Acceptance Criteria
- [ ] `map_yfinance_to_metadata()` function in `services/universe.py`
- [ ] Resolves `ticker_info.sector` via `SECTOR_ALIASES` → `GICSSector | None`
- [ ] Resolves `ticker_info.industry` via `INDUSTRY_GROUP_ALIASES` → `GICSIndustryGroup | None`
- [ ] Uses `classify_market_cap()` for market cap tier from `ticker_info.market_cap`
- [ ] Preserves raw strings in `raw_sector` and `raw_industry` fields
- [ ] Logs unmapped sector strings at WARNING level
- [ ] Logs unmapped industry strings at WARNING level
- [ ] Sets `last_updated` to `datetime.now(UTC)`
- [ ] Function is a module-level standalone (not a method on any class)

## Technical Details
- Co-locate with existing `build_sector_map()`/`build_industry_group_map()` in `services/universe.py`
- Alias resolution pattern: `SECTOR_ALIASES.get(ticker_info.sector.strip().lower())` → `GICSSector | None`
- Industry resolution: `INDUSTRY_GROUP_ALIASES.get(ticker_info.industry.strip().lower())` → `GICSIndustryGroup | None`
- Use `classify_market_cap()` from the same module (already a public method on `UniverseService`)
  - Since this is a standalone function (not on a class), import `_classify_market_cap` from `market_data` or extract the classification logic inline
- Log at WARNING: `"Unmapped yfinance sector for %s: %s"` and `"Unmapped yfinance industry for %s: %s"`
- Handle "Unknown" sector/industry gracefully (don't log warning for "Unknown" — that's the expected default)

### Edge Cases
- Sector = "Unknown" → None sector, no warning
- Industry = "Unknown" → None industry_group, no warning
- Sector = "" (empty string) → None sector
- market_cap = None → market_cap_tier = None
- market_cap = 0 → MarketCapTier.MICRO
- Sector with extra whitespace → trimmed before lookup
