# Task: Greeks Aggregation Module

## Description
Create `pricing/spreads.py` with `aggregate_spread_greeks()` that combines Greeks across all legs of a multi-leg spread. Short legs negate their Greeks, long legs add. Returns `OptionGreeks` or `None` if any leg is missing Greeks. This is the only pricing module function needed for spreads — construction logic lives in `scoring/`.

## Technical Details

### Function Signature
```python
def aggregate_spread_greeks(legs: list[SpreadLeg]) -> OptionGreeks | None:
    """Aggregate Greeks across spread legs with sign convention.

    LONG legs: add Greeks as-is.
    SHORT legs: negate all Greeks (delta, gamma, theta, vega, rho).
    Quantity multiplier: multiply each Greek by leg.quantity.

    Returns None if any leg has contract.greeks == None.
    Returns None if legs list is empty.
    """
```

### Sign Convention
- `PositionSide.LONG`: sign = +1
- `PositionSide.SHORT`: sign = -1
- Each Greek = sum(sign * quantity * greek for each leg)

### Second-Order Greeks
- Aggregate vanna, charm, vomma if ALL legs have them (not None)
- If any leg is missing second-order, set all three to None on result

### pricing_model on Result
- Use first leg's `contract.greeks.pricing_model`
- All legs in a spread should use the same pricing model

### Delta Bounds
- Aggregated delta for standard strategies (vertical, iron condor, straddle, strangle) stays within [-1, 1]
- If aggregate delta falls outside [-1, 1] due to unusual leg combinations, clamp to [-1, 1] with a log warning

### Module Boundary
- Imports: `models/` only (SpreadLeg, OptionGreeks, PositionSide, PricingModel)
- Cannot import: services, pandas, scoring, bsm, american

### Edge Cases
- Empty legs list: returns None
- Single leg LONG: returns same Greeks
- Single leg SHORT: returns negated Greeks
- All second-order present: aggregated
- Any second-order missing: all set to None
- Aggregate gamma < 0 (short-heavy spread): validator issue — use model_construct or adjust validation
- Aggregate delta outside [-1, 1]: clamped with warning
