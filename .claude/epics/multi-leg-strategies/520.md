# Task: Strategy Construction + Selection Engine

## Description
Create `scoring/spreads.py` with four strategy construction functions (`build_vertical_spread`, `build_iron_condor`, `build_straddle`, `build_strangle`) and a `select_strategy()` selection engine that uses IV regime, direction, and confidence to pick the optimal strategy. Each builder computes P&L mechanics, breakevens, and probability of profit (PoP). Returns `SpreadAnalysis | None` — graceful fallback to single-contract when a spread cannot be built.

### P&L Formulas
**Vertical Credit Spread** (sell higher premium, buy lower):
- net_premium = short_leg.mid - long_leg.mid (credit received)
- max_profit = net_premium
- max_loss = strike_width - net_premium
- breakeven: short_strike ± net_premium (depending on call/put)

**Vertical Debit Spread** (buy higher premium, sell lower):
- net_premium = long_leg.mid - short_leg.mid (debit paid)
- max_profit = strike_width - net_premium
- max_loss = net_premium
- breakeven: long_strike ± net_premium

**Iron Condor** (bear call spread + bull put spread):
- net_premium = total credit from both spreads
- max_profit = net_premium
- max_loss = wing_width - net_premium
- breakevens: [lower_short_strike - net_credit_put_side, upper_short_strike + net_credit_call_side]

**Straddle** (ATM call + ATM put):
- net_premium = call.mid + put.mid (total debit)
- max_profit = unlimited (use Decimal("999999.99") sentinel)
- max_loss = net_premium
- breakevens: [strike - net_premium, strike + net_premium]

**Strangle** (OTM call + OTM put):
- net_premium = call.mid + put.mid (total debit)
- max_profit = unlimited
- max_loss = net_premium
- breakevens: [put_strike - net_premium, call_strike + net_premium]

### PoP Estimation
Use BSM N(d2) approach:
```python
from scipy.stats import norm
# For credit spreads: probability price stays between breakevens
# For debit spreads: probability price moves beyond breakeven
d2 = (math.log(spot / breakeven) + (r - 0.5 * sigma**2) * T) / (sigma * math.sqrt(T))
pop = norm.cdf(d2)  # or 1 - norm.cdf(d2) depending on direction
```

### Decision Tree (select_strategy)
```python
vol_regime = classify_vol_regime(iv_rank)  # from indicators/iv_analytics.py
match (vol_regime, direction, confidence):
    case (VolRegime.ELEVATED | VolRegime.EXTREME, SignalDirection.NEUTRAL, _):
        return build_iron_condor(...)
    case (VolRegime.ELEVATED | VolRegime.EXTREME, _, _) if confidence < 0.4:
        return build_strangle(...)
    case (VolRegime.ELEVATED | VolRegime.EXTREME, _, _):
        return build_vertical_spread(...)  # credit spread
    case (VolRegime.LOW, SignalDirection.BULLISH | SignalDirection.BEARISH, _):
        return build_vertical_spread(...)  # debit spread
    case _:
        return None  # mid-IV or no IV data → single contract
```

### Edge Cases
- Insufficient contracts for any strategy: returns None
- Zero-width strikes (same strike): returns None
- Missing Greeks on contracts: returns None for net_greeks, still builds P&L
- iv_rank is None: select_strategy returns None
- All contracts same type (only calls or only puts): iron condor returns None
- Very wide bid-ask spread on leg: still builds, PoP may be low
