# Task: Second-Order Greeks

## Description
Extend the pricing module with second-order Greeks: vanna (dDelta/dsigma), charm (dDelta/dT), and vomma (dVega/dsigma). BSM gets analytical closed-form formulas reusing the existing `_d1_d2()` helper. BAW gets finite-difference cross-bumps using existing bump constants. Add `option_second_order_greeks()` dispatch function. Extend `OptionGreeks` model with 3 optional fields.

## Technical Details

### BSM Analytical Formulas (Merton 1973 extension)
Using existing `_d1_d2(S, K, T, r, q, sigma)` helper from `bsm.py`:
- **Vanna**: `-e^(-qT) * n(d1) * d2 / sigma` (same for calls and puts)
- **Charm (call)**: `-e^(-qT) * [n(d1) * (2(r-q)T - d2*sigma*sqrt(T)) / (2T*sigma*sqrt(T))] + q*e^(-qT)*N(d1)`
  - Put charm: similar with sign adjustments on N() terms
- **Vomma**: `vega * d1 * d2 / sigma` where `vega = S*e^(-qT)*n(d1)*sqrt(T)`

### BAW Finite-Difference Cross-Bumps
Using existing bump constants: `_DS_FRACTION=0.01`, `_DSIGMA=0.487`, `_DT=1/365`
- **Vanna**: `[Delta(sigma+dsigma) - Delta(sigma-dsigma)] / (2*dsigma)` — centered diff of delta w.r.t. sigma
- **Charm**: `[Delta(T-dT) - Delta(T)] / dT` — forward diff of delta w.r.t. time (guard T-dT<=0)
- **Vomma**: `[nu(sigma+dsigma) - 2*nu(sigma) + nu(sigma-dsigma)] / (dsigma^2)` — second derivative of price w.r.t. sigma

Each requires calling `american_greeks()` or `american_price()` with bumped params (12 extra evaluations total for 3 Greeks).

### Return Type
Returns `OptionGreeks` with ONLY vanna/charm/vomma populated (delta/gamma/theta/vega/rho set to 0.0). Caller merges with first-order Greeks via single construction.

Actually — return a simple tuple or NamedTuple of `(vanna, charm, vomma)` to avoid constructing a partial OptionGreeks. The caller in `scoring/contracts.py` will construct a single complete OptionGreeks with all fields. Use a `SecondOrderGreeks` NamedTuple.

### Edge Cases
- T approaching 0 -> BSM vanna/charm/vomma should degrade gracefully (return 0 or small value)
- sigma approaching 0 -> handle division by sigma in formulas
- Deep ITM/OTM -> values approach 0 for vanna/vomma
- BAW charm when T < dT -> forward difference fallback
- q=0 -> BAW call degenerates to BSM call (cross-verification)
