# Task: Vol Surface Analytics

## Description
Create `indicators/vol_surface.py` with tiered implied volatility surface analytics. Primary entry point `compute_vol_surface()` returns a `VolSurfaceResult` NamedTuple. Tier 1 fits a surface via `SmoothBivariateSpline` when >=6 contracts across >=2 expirations; Tier 2 falls back to standalone calculations for sparse chains. Includes skew_25d (25-delta risk reversal), smile curvature, ATM IV term structure, and Breeden-Litzenberger implied probability. **This file is shared with the `volatility-intelligence` epic** — we create it, they extend it.

## Technical Details

### Module Rules
- `indicators/` module: pure math, pandas/numpy in, float/NamedTuple out
- No Pydantic models, no API calls, no imports from `services/` or `models/`
- Use `numpy` and `scipy.interpolate` for fitting
- Contracts are passed as structured data (arrays of strikes, IVs, DTEs, option types)

### VolSurfaceResult NamedTuple
```python
class VolSurfaceResult(NamedTuple):
    skew_25d: float | None
    smile_curvature: float | None
    prob_above_current: float | None
    atm_iv_30d: float | None
    atm_iv_60d: float | None
    fitted_ivs: np.ndarray | None       # None on standalone fallback
    residuals: np.ndarray | None        # None on standalone fallback
    z_scores: np.ndarray | None         # None on standalone fallback
    r_squared: float | None             # None on standalone fallback
    is_1d_fallback: bool
    is_standalone_fallback: bool
```

### Input Format
Since `indicators/` cannot import Pydantic models, the public function accepts numpy arrays:
```python
def compute_vol_surface(
    strikes: np.ndarray,
    ivs: np.ndarray,
    dtes: np.ndarray,
    option_types: np.ndarray,  # 1.0 for call, -1.0 for put
    spot: float,
    risk_free_rate: float = 0.05,
) -> VolSurfaceResult:
```

### Tiered Logic
1. Count unique DTEs and total contracts
2. If >=6 contracts AND >=2 unique DTEs -> Tier 1 (fitted surface)
3. Else -> Tier 2 (standalone fallback, `is_standalone_fallback=True`)

### Breeden-Litzenberger (1978)
Implied risk-neutral density from butterfly spread: `f(K) ~ e^(rT) * [C(K-dK) - 2C(K) + C(K+dK)] / dK^2`
Integrate CDF -> `prob_above_current = 1 - CDF(spot)`. Approximate with trapezoidal integration from available strikes.

### Edge Cases
- All contracts same strike -> degenerate, returns None
- NaN in IVs -> filtered before fitting
- Zero IV -> excluded from surface
- Single expiration with many contracts -> Tier 2 (not enough DTE diversity)
- Very wide strike range -> spline extrapolation bounded
- All puts or all calls -> skew_25d may be None
