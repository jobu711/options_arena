## Technical Details

### `models/enums.py` — Add SentimentLabel

```python
class SentimentLabel(StrEnum):
    BULLISH = "bullish"

```

### `models/openbb.py` — New File (5 Models)

All models: `ConfigDict(frozen=True)`, field validators with `math.isfinite()`, UTC datetime validators.

**FundamentalSnapshot**: ticker, pe_ratio (`float | None`), forward_pe, peg_ratio, price_to_book, debt_to_equity, revenue_growth, profit_margin, market_cap (`int | None`), fetched_at (UTC datetime).

**UnusualFlowSnapshot**: ticker, net_call_premium (`float | None`), net_put_premium, call_volume (`int | None`), put_volume, put_call_ratio (`float | None`), fetched_at (UTC datetime).

**NewsHeadline**: title (str), published_at (`datetime | None`), sentiment_score (float, -1.0 to 1.0), source (`str | None`).

**NewsSentimentSnapshot**: ticker, headlines (`list[NewsHeadline]`), aggregate_sentiment (float, -1.0 to 1.0), sentiment_label (`SentimentLabel`), article_count (int), fetched_at (UTC datetime).

**OpenBBHealthStatus**: openbb_available (bool), yahoo_fundamentals (bool), stockgrid_flow (bool), last_checked (UTC datetime).

### `models/config.py` — OpenBBConfig

```python
class OpenBBConfig(BaseModel):
    enabled: bool = True
    fundamentals_enabled: bool = True
    unusual_flow_enabled: bool = True
    news_sentiment_enabled: bool = True
    fundamentals_cache_ttl: int = 3600    # 1 hour
    flow_cache_ttl: int = 300             # 5 minutes
    news_cache_ttl: int = 900             # 15 minutes
    request_timeout: int = 15             # seconds
    max_retries: int = 2
```

Add `openbb: OpenBBConfig = OpenBBConfig()` to `AppSettings`.

### `pyproject.toml`

Add under `[project.optional-dependencies]`:

```toml
openbb = ["openbb>=4.0", "vaderSentiment>=3.3"]
```

### Edge Cases
- All optional float fields as `None` → valid construction
- NaN/Inf in any float field → `ValidationError`
- Naive datetime (no tzinfo) → `ValidationError`
- Non-UTC datetime → `ValidationError`
- Sentiment score at exact bounds (-1.0 and 1.0) → valid
- Empty headlines list → valid (no articles found)

### Mocking Strategy
- No mocking needed — pure model/config tests
- For `OpenBBConfig` env override test: use `monkeypatch.setenv("ARENA_OPENBB__ENABLED", "false")`
