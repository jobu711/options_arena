# Task: Agent Prompt Enrichment

## Description

Add 3 new context sections to `render_context_block()` in `agents/_parsing.py`: Fundamental Profile, Unusual Options Flow, and News Sentiment. Each section is omitted entirely when its data is `None` — agents never see empty sections.

## Acceptance Criteria

- [ ] `render_context_block()` renders "Fundamental Profile" section when any fundamental field is not None
- [ ] `render_context_block()` renders "Unusual Options Flow" section when any flow field is not None
- [ ] `render_context_block()` renders "News Sentiment" section when sentiment data is not None
- [ ] Each section omitted entirely when all its fields are `None`
- [ ] Sections use `_render_optional()` helper for consistent None-safe formatting
- [ ] Section placement: after existing indicators/Greeks sections, before earnings warning
- [ ] ~15 tests covering all rendering scenarios
- [ ] `ruff check .`, `mypy --strict src/`, `pytest tests/ -v` all pass

## Technical Details

### New Sections in `render_context_block()` (`agents/_parsing.py`)

Insert after Greeks/contract mid section (around line 418), before earnings warning (around line 421):

```python
# --- Fundamental Profile ---
fundamental_lines = [
    _render_optional("P/E", ctx.pe_ratio, ".1f"),
    _render_optional("Forward P/E", ctx.forward_pe, ".1f"),
    _render_optional("PEG", ctx.peg_ratio, ".2f"),
    _render_optional("P/B", ctx.price_to_book, ".2f"),
    _render_optional("Debt/Equity", ctx.debt_to_equity, ".2f"),
    _render_optional("Revenue Growth (YoY)", ctx.revenue_growth, ".1%"),
    _render_optional("Profit Margin", ctx.profit_margin, ".1%"),
]
fundamental_lines = [ln for ln in fundamental_lines if ln is not None]
if fundamental_lines:
    lines.append("")
    lines.append("## Fundamental Profile")
    lines.extend(fundamental_lines)

# --- Unusual Options Flow ---
flow_lines = [
    _render_optional("Net Call Premium ($)", ctx.net_call_premium, ",.0f"),
    _render_optional("Net Put Premium ($)", ctx.net_put_premium, ",.0f"),
    _render_optional("Options Put/Call Ratio", ctx.options_put_call_ratio, ".2f"),
]
flow_lines = [ln for ln in flow_lines if ln is not None]
if flow_lines:
    lines.append("")
    lines.append("## Unusual Options Flow")
    lines.extend(flow_lines)

# --- News Sentiment ---
if ctx.news_sentiment is not None:
    lines.append("")
    lines.append("## News Sentiment")
    label = ctx.news_sentiment_label or "neutral"
    lines.append(f"Aggregate: {label.title()} ({ctx.news_sentiment:+.2f})")
    if ctx.recent_headlines:
        for headline in ctx.recent_headlines[:5]:
            lines.append(f"- \"{headline}\"")
```

### Format Considerations

- Revenue growth and profit margin: use `%` format (e.g., "12.8%")
- Net premium: use comma-separated format (e.g., "$4,200,000")
- Sentiment score: use signed format (e.g., "+0.42" or "-0.31")
- Headlines: quoted strings, max 5

### `_render_optional()` Reuse

The existing helper handles None and non-finite values. Use it for all float fields. For `revenue_growth` and `profit_margin`, note that the `%` format specifier multiplies by 100 automatically — verify whether the source data is already a percentage (0.128 = 12.8%) or raw (12.8 = 12.8%).
