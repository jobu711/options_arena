# Task: Verification gate — ruff + mypy + pytest all green

## Description
Run all three verification commands and fix any issues. This is the final quality gate before the epic can be marked complete. All linting, type checking, and tests must pass cleanly.

## Acceptance Criteria
- [ ] `uv run ruff check . --fix && uv run ruff format .` — zero errors, zero warnings
- [ ] `uv run pytest tests/ -v` — all ~150 tests pass, zero failures, zero errors
- [ ] `uv run mypy src/ --strict` — zero errors
- [ ] No `type: ignore` comments unless absolutely necessary (and documented why)
- [ ] No `# noqa` comments unless absolutely necessary (and documented why)
- [ ] Test coverage for models/ is >95%
- [ ] All imports work: `from options_arena.models import <every_exported_name>`
- [ ] All imports work: `from options_arena.utils import <every_exception>`

## Technical Details
- Run all three commands in sequence — each must pass independently
- Fix ruff issues first (formatting affects mypy line numbers)
- Fix mypy issues second (type errors may cause test failures)
- Fix test failures last
- Common mypy issues to watch for:
  - Missing return type annotations on test helpers
  - `Decimal` vs `float` type narrowing
  - `datetime.date.today()` in computed fields may need `# type: ignore[override]`
- If a fix in one area breaks another, iterate until all three pass simultaneously
