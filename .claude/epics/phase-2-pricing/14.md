# Task: Implement BSM pricing module

## Description
Implement `src/options_arena/pricing/bsm.py` — the Black-Scholes-Merton European pricing
module extended with continuous dividend yield (Merton 1973). This is the foundation that
BAW builds upon, so it must be correct and complete before any other pricing work begins.

Covers PRD requirement **FR-P1**.

- [ ] `bsm_greeks(S, K, T, r, q, sigma, option_type)` returns `OptionGreeks` with all 5 Greeks and `pricing_model=PricingModel.BSM`
- [ ] `bsm_vega(S, K, T, r, q, sigma)` returns standalone vega (for Newton-Raphson `fprime`)

## Technical Details

### Functions to implement

1. **`bsm_price`** — Core pricing:
   ```
   d1 = (ln(S/K) + (r - q + sigma²/2) * T) / (sigma * sqrt(T))
   d2 = d1 - sigma * sqrt(T)
   Call = S * e^(-qT) * N(d1) - K * e^(-rT) * N(d2)
   Put  = K * e^(-rT) * N(-d2) - S * e^(-qT) * N(-d1)
   ```

2. **`bsm_greeks`** — All analytical Greeks:
   - Delta: `e^(-qT) * N(d1)` (call), `-e^(-qT) * N(-d1)` (put)
   - Gamma: `e^(-qT) * n(d1) / (S * sigma * sqrt(T))`
   - Theta: call/put specific formulas (see `pricing/CLAUDE.md`)
   - Vega: `S * e^(-qT) * n(d1) * sqrt(T)`
   - Rho: `K * T * e^(-rT) * N(d2)` (call), `-K * T * e^(-rT) * N(-d2)` (put)
   - Returns `OptionGreeks(pricing_model=PricingModel.BSM, ...)`

3. **`bsm_vega`** — Standalone for IV solver `fprime`. Same formula as vega in greeks.

4. **`bsm_iv`** — Newton-Raphson loop:
   - Initial guess from `initial_guess` param (default 0.30)
   - Each iteration: `sigma -= (bsm_price(sigma) - market_price) / bsm_vega(sigma)`
   - Clamp sigma to [1e-6, 5.0] each step
   - Converge when `abs(price_diff) < tol`
   - Max iterations from `PricingConfig.iv_solver_max_iter`
