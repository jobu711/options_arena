# Task: FRED Service

## Description
Build `fred.py` — fetches the 10-year Treasury yield from FRED API as a proxy for the risk-free rate. Gracefully falls back to `PricingConfig.risk_free_rate_fallback` on any error. Never raises — the risk-free rate should never block the pipeline.

## Acceptance Criteria
- [ ] `FredService` class with DI constructor (`ServiceConfig`, `PricingConfig`, `ServiceCache`)
- [ ] `fetch_risk_free_rate()` returns `float` (decimal fraction: 0.045 = 4.5%)
- [ ] Fetches from FRED API: `series_id=DGS10`, `sort_order=desc`, `limit=1`, `file_type=json`
- [ ] Percentage-to-decimal conversion: FRED `4.5` → `0.045`
- [ ] Falls back to `PricingConfig.risk_free_rate_fallback` when:
  - FRED API key is missing (`ServiceConfig.fred_api_key is None`)
  - Network error / timeout
  - Malformed response / parse failure
- [ ] Logs WARNING on fallback (never raises)
- [ ] Uses `httpx.AsyncClient` with configurable timeout
- [ ] Caches successful responses for 24 hours (via `ServiceCache`)
- [ ] `close()` closes httpx client
- [ ] ~8 tests covering all paths

## Technical Details
- **Files to create**: `src/options_arena/services/fred.py`
- **Test files**: `tests/unit/services/test_fred.py`
- FRED API endpoint: `https://api.stlouisfed.org/fred/series/observations`
- Query params: `series_id=DGS10`, `sort_order=desc`, `limit=1`, `file_type=json`, `api_key={key}`
- Response parsing: most recent non-null observation `value` field
- httpx timeout: `httpx.Timeout(connect=5.0, read=config.fred_timeout)`
- Requires `ARENA_SERVICE__FRED_API_KEY` env var (or `ServiceConfig.fred_api_key`)
