# Task: Market Data Service

## Description
Build `market_data.py` — the most complex service module. Fetches and normalizes all yfinance market data (OHLCV, quotes, ticker info) into typed Pydantic models. Includes the 3-tier dividend yield extraction waterfall (FR-M7.1), MarketCapTier classification, and cross-validation logic.

## Acceptance Criteria
- [ ] `MarketDataService` class with DI constructor (`ServiceConfig`, `ServiceCache`, `RateLimiter`)
- [ ] `fetch_ohlcv()`: yfinance history → `list[OHLCV]`, cache-first, sorted by date ascending
- [ ] `fetch_ohlcv()` raises `TickerNotFoundError` for invalid tickers
- [ ] `fetch_ohlcv()` raises `InsufficientDataError` for empty yfinance responses
- [ ] `fetch_quote()`: yfinance fast_info → `Quote` with UTC timestamp
- [ ] `fetch_ticker_info()`: yfinance info → `TickerInfo` with full dividend waterfall
- [ ] Dividend waterfall tier 1: `info["dividendYield"]` → `DividendSource.FORWARD`
- [ ] Dividend waterfall tier 2: `info["trailingAnnualDividendYield"]` → `DividendSource.TRAILING`
- [ ] Dividend waterfall tier 3: `sum(get_dividends("1y")) / price` → `DividendSource.COMPUTED`
- [ ] Dividend waterfall tier 4: `0.0` → `DividendSource.NONE`
- [ ] Fall-through condition is `value is None`, NOT falsy — `0.0` is valid data
- [ ] Cross-validation: warn if `abs(yield - rate/price) / max(yield, 1e-9) > 0.20`
- [ ] Audit fields populated: `dividend_rate`, `trailing_dividend_rate`
- [ ] `MarketCapTier` classification with correct thresholds (mega ≥ 200B, large ≥ 10B, mid ≥ 2B, small ≥ 300M, micro < 300M)
- [ ] `fetch_batch_ohlcv()`: concurrent fetch with per-ticker error isolation via `asyncio.gather(return_exceptions=True)`
- [ ] All yfinance calls wrapped via `asyncio.to_thread()` + `asyncio.wait_for(timeout)`
- [ ] ~22 tests covering all paths

## Technical Details
- **Files to create**: `src/options_arena/services/market_data.py`
- **Test files**: `tests/unit/services/test_market_data.py`
- yfinance keys are **camelCase** (Context7-verified): `dividendYield`, `trailingAnnualDividendYield`, `dividendRate`, `trailingAnnualDividendRate`, `marketCap`, `sector`, `shortName`
- `_yf_call()` internal helper wraps all sync yfinance calls with to_thread + timeout + error mapping
- `_extract_dividend_yield()` is a pure function (testable independently)
- `_classify_market_cap()` is a pure function
- `fetch_batch_ohlcv()` returns `dict[str, list[OHLCV] | DataFetchError]` — exceptions as values
