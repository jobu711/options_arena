## Technical Details
- `__init__.py` follows the re-export pattern from `models/__init__.py` and `services/__init__.py`
- `PRAGMA table_info(table_name)` returns rows with: `cid, name, type, notnull, dflt_value, pk`

- `PRAGMA foreign_key_list(table_name)` returns rows with: `id, seq, table, from, to, on_update, on_delete, match`
