# Task: Pipeline Phase 1 + Phase 2

## Description
Create `src/options_arena/scan/pipeline.py` with `ScanPipeline` class: constructor (DI for all services), `run()` skeleton with cancellation checks between phases, `_phase_universe()` (Phase 1), and `_phase_scoring()` (Phase 2).

## Acceptance Criteria
- [ ] `ScanPipeline.__init__` accepts: `settings: AppSettings`, `market_data: MarketDataService`, `options_data: OptionsDataService`, `fred: FredService`, `universe: UniverseService`, `repository: Repository`
- [ ] `run()` accepts: `preset: ScanPreset`, `token: CancellationToken`, `progress: ProgressCallback` → returns `ScanResult`
- [ ] `run()` checks `token.is_cancelled` between each phase, returns partial `ScanResult` with `cancelled=True`
- [ ] `_phase_universe()`: fetches optionable tickers, filters by SP500 preset, batch OHLCV, filters by `ohlcv_min_bars`, reports progress
- [ ] `_phase_scoring()`: converts OHLCV→DataFrame, computes indicators, retains raw signals, calls `score_universe`, calls `determine_direction` with RAW values (NOT normalized), sets direction on each `TickerScore`
- [ ] Direction uses raw signals: `adx=raw.adx or 0.0`, `rsi=raw.rsi or 50.0`, `sma_alignment=raw.sma_alignment or 0.0`
- [ ] ~20 tests written and passing
- [ ] ruff + mypy --strict pass

## Technical Details
- Phase 1 uses: `universe.fetch_optionable_tickers()`, `universe.fetch_sp500_constituents()`, `market_data.fetch_batch_ohlcv()`
- Phase 2 uses: `ohlcv_to_dataframe()`, `compute_indicators()`, `score_universe()` from `scoring/normalization`, `determine_direction()` from `scoring/direction`
- **Critical**: `score_universe()` normalizes signals to 0–100 percentile ranks on `TickerScore.signals`. `determine_direction()` needs ABSOLUTE values (ADX < 15 → NEUTRAL). Must pass raw signals, not normalized.
- Tests should mock all service calls. Verify: ohlcv_map filtering, min_bars enforcement, SP500 preset filtering, progress callback invocation order, raw_signals retention, direction classification from raw values, None fallback values for missing indicators, cancellation between Phase 1→2.
