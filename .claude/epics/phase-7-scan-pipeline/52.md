# Task: Indicator dispatch — indicators.py

## Description
Create `src/options_arena/scan/indicators.py` with the data-driven indicator dispatch system: `InputShape` StrEnum, `IndicatorSpec` NamedTuple, the 14-entry `INDICATOR_REGISTRY`, `ohlcv_to_dataframe()` converter, and `compute_indicators()` generic dispatch loop.

This replaces v3's 14 copy-paste indicator blocks with a single loop driven by a typed registry.

## Acceptance Criteria
- [ ] `InputShape` StrEnum with 5 members: `CLOSE`, `HLC`, `CLOSE_VOLUME`, `HLCV`, `VOLUME`
- [ ] `IndicatorSpec` is a `NamedTuple(field_name: str, func: Callable[..., pd.Series], input_shape: InputShape)`
- [ ] `INDICATOR_REGISTRY` has exactly 14 entries matching the epic's registry table
- [ ] All 14 `field_name` values match `IndicatorSignals` field names exactly
- [ ] 4 name mappings verified: `stoch_rsi→stochastic_rsi`, `atr_percent→atr_pct`, `obv_trend→obv`, `ad_trend→ad`
- [ ] `ohlcv_to_dataframe()` converts `list[OHLCV]` to DataFrame with float columns `[open, high, low, close, volume]`, date index, sorted ascending
- [ ] `compute_indicators()` dispatches via `match spec.input_shape`, uses `series.iloc[-1]`, converts NaN→None
- [ ] Isolated per-indicator `try/except` — one failure logs WARNING, sets field to `None`, continues
- [ ] ~25 tests written and passing
- [ ] ruff + mypy --strict pass

## Technical Details
- `field_name` on each spec must match an `IndicatorSignals` field — NOT the function name. 4 field names differ from their function names.
- `ohlcv_to_dataframe`: OHLCV uses `Decimal` prices — convert to `float` for indicator math. Sort by date ascending. Index on date.
- `compute_indicators`: For each `IndicatorSpec`, use `match spec.input_shape` to extract the right columns from the DataFrame, call `spec.func(...)`, take `series.iloc[-1]`. If the value is NaN (via `math.isnan`), set to `None`.
- Per-indicator isolation: catch `Exception` (not bare `except:`), log `logging.warning(...)`, set field to `None`, continue loop.
- Tests should cover: registry completeness (14 entries), function reference correctness, field name matches `IndicatorSignals`, InputShape exhaustive enum, column dispatch match, `ohlcv_to_dataframe` (Decimal→float, date index, column names, sort order), `compute_indicators` (happy path all 14, isolated failure, NaN→None, empty DataFrame, InsufficientDataError).
