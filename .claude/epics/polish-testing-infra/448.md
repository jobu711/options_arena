# Task: Consolidate conftest fixtures using factories

## Description

Consolidate scattered test fixtures by migrating shared fixtures to the root `tests/conftest.py`
using the new factory functions. Remove duplicates across module conftest files, delete the
empty scan conftest, and update imports in scoring tests to use the new factory location.

## Acceptance Criteria

- [ ] `make_contract()` removed from `tests/unit/scoring/conftest.py` ‚Äî replaced by `make_option_contract()` from `tests/factories.py`
- [ ] All scoring tests updated to import from `tests.factories` instead of local conftest
- [ ] Shared config fixtures moved to root `tests/conftest.py` (if duplicates found)
- [ ] Factory-based convenience fixtures added to root conftest (e.g., `sample_contract`, `sample_quote`)
- [ ] Empty `tests/unit/scan/conftest.py` deleted
- [ ] No fixture name collisions across conftest files
- [ ] Full test suite passes after all changes

## Technical Details

### Files Affected
- `tests/conftest.py` ‚Äî populate with shared fixtures
- `tests/unit/scoring/conftest.py` ‚Äî remove `make_contract()`, update to use factories
- `tests/unit/scan/conftest.py` ‚Äî delete (empty, 1 line)
- `tests/unit/scoring/test_*.py` ‚Äî update imports from `conftest.make_contract` to `tests.factories.make_option_contract`
- `tests/unit/agents/conftest.py` ‚Äî review for factory opportunities (keep agent-specific fixtures)
- `tests/unit/api/conftest.py` ‚Äî review (keep API-specific DI mocks)

### Implementation Steps
1. Remove `make_contract()` from `scoring/conftest.py`
2. Update all scoring test imports to use `from tests.factories import make_option_contract`
3. Audit remaining conftest files for shared/duplicated fixtures
4. Move any shared fixtures to root `tests/conftest.py`, backed by factory calls
5. Add convenience fixtures to root conftest: `sample_contract`, `sample_quote`, `sample_market_context`
6. Delete `tests/unit/scan/conftest.py`
7. Run full test suite to verify no regressions

### Key Considerations
- Keep domain-specific fixtures in their module conftest (agent mocks, API DI overrides)
- Only move fixtures that are truly shared or duplicated
- Convenience fixtures should use factories internally: `@pytest.fixture def sample_contract(): return make_option_contract()`
- Watch for fixture name collisions between root and module conftest files
