# Task: Scoring cutoffs in pipeline orchestrator

## Description
Add `min_score` and `min_direction_confidence` cutoffs in the pipeline orchestrator (`scan/pipeline.py`) post-Phase 2, alongside the existing `direction_filter`. When `ScoringFilters.min_score > 0` or `ScoringFilters.min_direction_confidence > 0`, drop tickers below threshold before Phase 3 begins. Log the filtering counts.

This is a new feature — tickers scoring below the threshold will be eliminated before the expensive Phase 3 chain fetching.

## Acceptance Criteria
- [ ] `min_score` cutoff applied post-Phase 2, before Phase 3
- [ ] `min_direction_confidence` cutoff applied post-Phase 2, before Phase 3
- [ ] Cutoffs applied AFTER existing `direction_filter` (preserve ordering)
- [ ] Logging: before/after counts for each cutoff applied
- [ ] `min_score=0.0` (default) → no filtering (no regression)
- [ ] `min_direction_confidence=0.0` (default) → no filtering (no regression)
- [ ] CLI `--min-confidence` arg exposed (note: `--min-score` already exists)
- [ ] API `ScanRequest` gains `min_direction_confidence` field
- [ ] ~10 new tests

## Technical Details

### Pipeline orchestrator (`scan/pipeline.py`):
After the existing direction filter block (lines ~147-158), add:

```python
# Post-Phase 2: Apply scoring cutoffs
scoring_filters = self._settings.scan.filters.scoring

# Direction filter (existing, now reads from scoring_filters)
if scoring_filters.direction_filter is not None:
    before = len(scoring_result.scores)
    scoring_result.scores = [
        ts for ts in scoring_result.scores
        if ts.direction == scoring_filters.direction_filter
    ]
    logger.info("Direction filter (%s): %d -> %d", ...)

# min_score cutoff (NEW)
if scoring_filters.min_score > 0.0:
    before = len(scoring_result.scores)
    scoring_result.scores = [
        ts for ts in scoring_result.scores
        if ts.composite_score >= scoring_filters.min_score
    ]
    logger.info("min_score cutoff (%.1f): %d -> %d", ...)

# min_direction_confidence cutoff (NEW)
if scoring_filters.min_direction_confidence > 0.0:
    before = len(scoring_result.scores)
    scoring_result.scores = [
        ts for ts in scoring_result.scores
        if ts.direction_confidence >= scoring_filters.min_direction_confidence
    ]
    logger.info("min_confidence cutoff (%.2f): %d -> %d", ...)
```
