# Task: Universe Service — 3 Fetch Methods + Preset-Info Endpoint

## Description
Add 3 new fetch methods to `UniverseService` following the existing cache-first, 24h TTL, never-raises pattern: `fetch_nasdaq100_constituents()`, `fetch_russell2000_tickers()`, `fetch_most_active()`. Add `GET /api/universe/preset-info` endpoint returning `list[PresetInfo]` via `asyncio.gather`.

## Acceptance Criteria
- [ ] `fetch_nasdaq100_constituents()` returns ~100 tickers from GitHub CSV with CBOE cross-ref + curated fallback
- [ ] `fetch_russell2000_tickers()` queries metadata index for `market_cap_tier in {SMALL, MICRO}` with CBOE cross-ref
- [ ] `fetch_most_active()` returns curated seed list (~250 names) with CBOE cross-ref
- [ ] All 3 methods follow cache-first, 24h TTL, never-raises contract (return empty list on failure)
- [ ] `GET /api/universe/preset-info` returns `list[PresetInfo]` with counts for all 6 presets
- [ ] Endpoint uses `asyncio.gather` for parallel count fetches
- [ ] All tests pass

## Technical Details
- **`services/universe.py`** (line 153-400): Add 3 new methods following existing pattern:
  - `fetch_nasdaq100_constituents()`: GitHub CSV → parse → CBOE cross-ref → curated fallback (~100 names). Same pattern as `fetch_sp500_constituents()`.
  - `fetch_russell2000_tickers()`: Query `Repository.get_all_ticker_metadata()` filtering `market_cap_tier in {SMALL, MICRO}` → CBOE cross-ref. Unique — uses metadata index instead of external CSV.
  - `fetch_most_active()`: Curated seed list hardcoded in module (same pattern as SP500 fallback list) → CBOE cross-ref. ~250 high-volume names.
- **`api/routes/universe.py`** (line 38-276): Add `GET /api/universe/preset-info` endpoint. Returns `list[PresetInfo]` (from Task 285's model). Uses `asyncio.gather` to call all 6 preset fetch methods in parallel, counts results, builds response.
- Cache keys: `NASDAQ100_CACHE_KEY`, `RUSSELL2000_CACHE_KEY`, `MOST_ACTIVE_CACHE_KEY` as module constants.
- Follow existing two-tier cache pattern (in-memory LRU + SQLite WAL).

### Edge Cases
- GitHub CSV source unavailable → curated fallback returned
- Metadata index empty → Russell 2000 returns empty list
- CBOE optionable set empty → all methods return empty list
- Cache expired → re-fetch triggered
- Network timeout → never-raises contract honored
