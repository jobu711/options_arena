## Technical Details

### Example Format
```python
TREND_SYSTEM_PROMPT = (
    """You are a trend analysis agent. ...

## Example Output
```json
{
  "agent_name": "trend",
  "direction": "bullish",
  "confidence": 0.55,
  "argument": "ACME shows strong upward momentum with ADX at 32.5 indicating a ...",
  "key_points": ["ADX above 25 threshold at 32.5", "SuperTrend bullish since ..."],
  "risks_cited": ["RSI approaching overbought at 68"],
  "contracts_referenced": ["ACME 150C 2026-04-18"],
  "model_used": "llama-3.3-70b-versatile"
}
```
"""
    + PROMPT_RULES_APPENDIX
)
```

### Domain-Specific Context Labels Per Agent
- **Trend**: ADX, SuperTrend, SMA crossovers, RSI momentum, MACD signal
- **Contrarian**: Prior agent consensus, agreement score, dissenting signals
- **Volatility**: IV RANK, IV PERCENTILE, ATM IV 30D, HV/IV ratio, term structure

### Key Constraints
- Groq/Llama-optimized formatting: JSON blocks, not markdown tables
- Examples must be realistic but anonymized (ACME ticker, plausible prices/Greeks)
- Confidence must be calibrated: 0.4-0.7 range demonstrates the prompt's calibration instructions
- Must leave headroom in 8192 context window (system prompt + context block + response)
