# Task: Create ServiceBase ABC with unit tests and docs

## Description
Create `services/base.py` containing the `ServiceBase(Generic[ConfigT])` mixin ABC that
unifies DI constructor, `close()` lifecycle, `_cached_fetch()`, `_retried_fetch()`, and
`_yf_call()`. Add comprehensive unit tests and update `services/__init__.py` re-exports
and `services/CLAUDE.md` documentation.

## Technical Details

### `services/base.py` Implementation
- `Generic[ConfigT]` with no bound — config types are heterogeneous BaseModel subclasses
- No `@abstractmethod` — mixin pattern, services opt into helpers they need
- `_cached_fetch` default: `model_type.model_validate_json(cached)` / `model.model_dump_json().encode()`
- `_cached_fetch` custom: optional `deserializer: Callable[[bytes], T]` overrides default deserialization
- `_retried_fetch` passes `self._limiter` automatically to `fetch_with_limiter_retry()`
- `_yf_call` maps all exceptions to `DataSourceUnavailableError`
- Import from: `services/cache.py`, `services/rate_limiter.py`, `services/helpers.py`, `utils/exceptions.py`

### Edge Cases
- Cache returns `None` (miss): factory must be called
- Cache returns corrupt bytes: `model_validate_json` raises, propagates
- Factory raises exception: propagates unchanged
- `_yf_call` with `DataFetchError` subclass: should NOT double-wrap
- `_retried_fetch` with `limiter=None`: should still work (pass None to helper)
- Empty `bytes` from cache: should handle gracefully

### Mocking Strategy
- `ServiceCache(config, db_path=None, max_size=100)` for in-memory cache
- `RateLimiter(rate=1000.0, max_concurrent=100)` for fast tests
- `unittest.mock.AsyncMock` for factory callables
- `unittest.mock.patch("asyncio.to_thread")` for `_yf_call` tests
- `unittest.mock.patch("asyncio.wait_for")` for timeout tests
