# Task: Smoothing Integration into compute_greeks

## Description
Modify `scoring/contracts.py` `compute_greeks()` to group contracts by (strike, expiration), call `smooth_iv_parity()` when both call and put exist at the same strike, and use the smoothed IV as sigma for Greeks computation. Set `greeks_source=GreeksSource.SMOOTHED` on smoothed contracts.

## Acceptance Criteria
- [ ] Contracts grouped by (strike, expiration) before Greeks computation
- [ ] `smooth_iv_parity()` called when both call and put exist at same strike/expiration
- [ ] Smoothed IV stored on contract via `model_copy(update={"smoothed_iv": smoothed})`
- [ ] Smoothed IV used as `sigma` input to `option_greeks()` instead of raw `market_iv`
- [ ] `greeks_source=GreeksSource.SMOOTHED` set on smoothed contracts
- [ ] When only one side exists at a strike, raw `market_iv` used (no smoothing)
- [ ] Smoothing skipped when `use_parity_smoothing=False` in config (backward compat)
- [ ] Tier 1 contracts (pre-computed Greeks) unchanged — smoothing only applies to Tier 2

## Technical Details
- Add optional `use_parity_smoothing: bool = True` parameter to `compute_greeks()` signature
- Before the existing per-contract loop, build a lookup: `pairs: dict[tuple[Decimal, date], dict[str, OptionContract]]` mapping `(strike, expiration)` → `{"call": contract, "put": contract}`
- In Tier 2 processing, before using `sigma = contract.market_iv`, look up the pair and call `smooth_iv_parity()` if both sides exist
- `smooth_iv_parity()` imported from `pricing.iv_smoothing` (allowed — `scoring/` can import from `pricing/`)
- Use `float(contract.bid)`, `float(contract.ask)` for spread computation (Decimal → float)

   - WHY: O(n) grouping to find call/put pairs at same strike/expiration
4. In Tier 2 block, after `sigma = contract.market_iv` (line 223), add smoothing:
   ```python
   smoothed_iv_value: float | None = None
   if use_parity_smoothing:
       key = (contract.strike, contract.expiration)
       pair = pairs.get(key, {})
       other_type = "put" if contract.option_type == OptionType.CALL else "call"
       other = pair.get(other_type)
       if other is not None and math.isfinite(other.market_iv) and other.market_iv > 0:
           smoothed_iv_value = smooth_iv_parity(
               call_iv=contract.market_iv if contract.option_type == OptionType.CALL else other.market_iv,
               put_iv=other.market_iv if contract.option_type == OptionType.CALL else contract.market_iv,
               call_bid=float(contract.bid if contract.option_type == OptionType.CALL else other.bid),
               call_ask=float(contract.ask if contract.option_type == OptionType.CALL else other.ask),
               put_bid=float(other.bid if contract.option_type == OptionType.CALL else contract.bid),
               put_ask=float(other.ask if contract.option_type == OptionType.CALL else contract.ask),
           )
           if math.isfinite(smoothed_iv_value) and smoothed_iv_value > 0:
               sigma = smoothed_iv_value
   ```
   - WHY: Use smoothed IV as sigma when pair exists, fall back to raw IV otherwise
5. In the `model_copy(update=...)` call (line 306-312), add `smoothed_iv` and update `greeks_source`:

   greeks_source = GreeksSource.SMOOTHED if smoothed_iv_value else GreeksSource.COMPUTED

       "greeks": greeks,
       "market_iv": sigma,
       "smoothed_iv": smoothed_iv_value,
       "greeks_source": greeks_source,
   })

   - WHY: Track smoothing provenance on each contract
