# Task: Project scaffold ‚Äî FastAPI + Vue + serve command

## Description

Set up both the FastAPI backend and Vue 3 SPA frontend from scratch. After this task,
`options-arena serve` starts the API and serves a hello-world Vue page. Dev mode runs
both servers with HMR.

### Backend scaffold (`src/options_arena/api/`)

- `__init__.py` ‚Äî re-exports `app` and `create_app`
- `app.py` ‚Äî app factory with `lifespan` (creates all services at startup, closes at shutdown),
  CORS middleware, StaticFiles mount for `web/dist/`
- `deps.py` ‚Äî `Depends()` providers: `get_repo()`, `get_market_data()`, `get_options_data()`,
  `get_fred()`, `get_universe()`, `get_settings()`, `get_operation_lock()`
- `routes/__init__.py` ‚Äî empty, router registration happens in `app.py`
- `schemas.py` ‚Äî placeholder (thin request/response wrappers added in later tasks)
- `CLAUDE.md` ‚Äî already exists, read before implementing

### Frontend scaffold (`web/`)

- `npm init` + install deps: `vue`, `vue-router`, `pinia`, `primevue`, `@primeuix/themes`,
  `primeicons`, `typescript`, `vite`, `@vitejs/plugin-vue`
- `vite.config.ts` ‚Äî dev proxy (`/api` ‚Üí `:8000`, `/ws` ‚Üí `:8000`), build to `dist/`
- `tsconfig.json` + `tsconfig.node.json`
- `index.html` ‚Äî Vite entry, `<body class="dark-mode">`
- `src/main.ts` ‚Äî createApp + PrimeVue (Aura dark) + ToastService + router + Pinia
- `src/App.vue` ‚Äî layout shell with nav (6 links) + `<router-view>` + Toast host
- `src/router/index.ts` ‚Äî 6 lazy-loaded routes (pages are placeholder stubs)
- `src/assets/variables.css` ‚Äî financial accent color CSS custom properties
- `src/types/index.ts` ‚Äî re-export placeholder
- Stub page files (one-liner `<template>` each): DashboardPage, ScanPage,
  ScanResultsPage, DebateResultPage, UniversePage, HealthPage
- `web/CLAUDE.md` ‚Äî already exists, read before implementing

### `options-arena serve` CLI command

- Add `serve` command to `cli/` (Typer)
- `--host`, `--port`, `--no-open`, `--reload` options
- Uses `uvicorn.run()` ‚Äî import uvicorn inside the function (lazy, optional dep)
- Auto-opens browser via `webbrowser.open()` unless `--no-open`

### Python dependencies

- `uv add fastapi uvicorn[standard]` ‚Äî add to main deps (or `[web]` optional group)
